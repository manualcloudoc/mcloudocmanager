# 윈도우 에이전트 설치 관련 주의사항

윈도우 에이전트 설치 파일은 운영 체제 플랫폼(64비트, 32비트)별 각각 제공됩니다. 사용하는 운영체제 플랫폼 정보를 확인 후 올바른 설치 파일을 설치해야 합니다.

에이전트 업데이트는 메이저 업그레이드(Major Upgrade) 방식으로 기존 버전을 제거하고 새 버전을 설치합니다. 이 과정에서 기존 버전에서 생성한 작업 데이터는 제거되지 않고 유지됩니다. 에이전트 설치는 궁극적으로 운영체제에서 제공하는 msiexec.exe와 윈도우 인스톨러(Window Installer) 서비스에 의해 수행됩니다. 커널 모듈 배포로 인해 설치 후 윈도우 시스템 재시작은 필수입니다. 설치 완료 단계에서 표시되는 재시작 메시지에서 **예**를 선택해야 합니다.  **​**

**아니오**를 선택하여 수동으로 시스템 재시작할 경우 **시작메뉴**, **전원**, **다시 시작** 메뉴로 진행 바랍니다. 윈도우의 빠른 시작 기능이 활성화된 환경에서 **시스템 종료** 메뉴를 선택하면 모듈 교체가 이루어지지 않아 설치가 완료되지 않습니다.

<div align="left"><img src="../../.gitbook/assets/img_001 (61).png" alt=""></div>

설치 진행 시 반드시 아래 위치에 로그가 생성되며, 설치 오류 발생 시 참조할 수 있습니다.

* 폴더 위치 : C:\NetID\Log\Setup
* 파일 이름 : ClouDoc\_setup\_<설치진행시간>.log
