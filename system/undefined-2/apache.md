# Apache 상태 서버 관제 로그 항목 및 비정상 판단 기준

서버 관제 로그중의 하나인 Apache 상태 로그에 대해 설명합니다.\
Apache 상태 로그는 Apache의 모듈중의 하나로 서버의 현재 상태와 성능 정보를 보여줍니다.

문서에서 경로는 리눅스 기준으로 명시하였습니다. 윈도우 서버인 경우에는 C: 드라이브 하위의 동일 경로입니다.

* **로그의 위치 : /plusdrive/log/monitoring\_log/apache\_status**
* **로그 파일 명 : server\_status\_년원일\_시분초.html**

### <mark style="color:$primary;">로그 항목별 상세 설명</mark>

현재 처리중인 요청 수, 연결 수, CPU 및 메모리 사용량, 로드 및 상태 정보가 표시됩니다.

| <p><strong>항목별 상세</strong></p><p></p><p><strong>Server Version</strong>: 현재 실행 중인 Apache 버전 정보 </p><p><strong>Server Built</strong>: Apache가 빌드된 날짜 정보 </p><p><strong>Current Time</strong>: 현재 시스템 시간 </p><p><strong>Restart Time</strong>: Apache가 마지막으로 다시 시작된 시간 </p><p><strong>Server uptime</strong>: Apache가 실행된 시간 </p><p><strong>Total accesses</strong>: 현재까지 처리된 요청의 총 수 </p><p><strong>CPU Usage</strong>: 현재 CPU 사용률 </p><p><strong>Requests per second</strong>: 초당 처리 가능한 요청 수 </p><p><strong>Bytes per second</strong>: 초당 전송 가능한 데이터 양 </p><p><strong>Bytes per request</strong>: 한 번의 요청에 대한 평균 데이터 양 </p><p><strong>Busy workers</strong>: 현재 활성화된 워커 수 </p><p><strong>Idle workers</strong>: 현재 대기 중인 워커 수 </p><p><strong>Server uptime</strong>: 서버가 실행된 시간 </p><p><strong>Server load</strong>: 현재 시스템의 부하 상태 </p><p><strong>Scoreboard</strong>: 현재 워커 상태 </p><p></p><p><strong>워커 상태를 나타내는 문자 별 설명</strong> </p><p></p><p>_: 워커가 초기화되지 않은 상태를 나타냅니다. </p><p>S: 워커가 대기 중인 상태를 나타냅니다. </p><p>R: 워커가 요청을 처리하는 중인 상태를 나타냅니다. </p><p>W: 워커가 요청을 기다리는 상태를 나타냅니다. </p><p>K: 워커가 종료되어 소멸된 상태를 나타냅니다. </p><p>D: 워커가 디스크 I/O 또는 네트워크 I/O를 처리하는 중인 상태를 나타냅니다. </p><p>C: 워커가 클로즈(Close)된 상태를 나타냅니다. 이는 워커가 현재 요청을 처리하고 있지 않으며, 클라이언트와 연결이 종료되었다는 것을 의미합니다. </p><p>L: 워커가 락(Locked)되어 있는 상태를 나타냅니다. 이는 워커가 현재 다른 프로세스에 의해 사용 중이라는 것을 의미합니다. </p><p>G: 워커가 그레이스풀(Graceful)한 종료 중인 상태를 나타냅니다. 이는 웹 서버의 종료 등 그레이스풀한 프로세스 종료 과정에서 워커가 실행되는 상태를 의미합니다. </p><p>I: 워커가 IDLE(유휴) 상태를 나타냅니다. 이는 워커가 대기 중이지만, 요청을 처리할 수 있는 상태라는 것을 의미합니다.</p> |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

로그 파일을 시간대별로 확인할 수 있으며, 현재 상태는 인터넷 웹 브라우저 주소창에 **서버주소/server-status** 입력하시면 확인할 수 있습니다. 그리고 **서버주소/server-status?refresh=3**과 같이 refresh 옵션(숫자 변경 가능)을 입력하면 3초마다 자동 새로 고침 되어 모니터링이 가능합니다.
