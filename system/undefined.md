# 시스템 요구 사항

### <mark style="color:$primary;">지원하는 서버 운영체제</mark>

서버를 설치할 수 있는 서버 운영체제(64-bit 권장)의 종류는 아래와 같습니다.

* **Windows 계열 운영체제**:

> - Windows server 2003 32-bit/64-bit
> - Windows server 2008 32-bit/64-bit
> - Windows server 2012 32-bit/64-bit

{% hint style="info" %}
데모용인 경우 Windows XP, Windows Vista, Windows 7등에도 설치 가능합니다.
{% endhint %}

* **Linux 계열 운영체제**:

> - CentOS
> - Fedora
> - Ubuntu
> - RedHat

* **유닉스(Unix)계열 운영체제**:

> - IBM AIX
> - SUN Solaris
> - HP-UNIX

{% hint style="info" %}
유닉스(Unix) 서버는 소스코드 빌드를 거쳐 주문하신 고객사에 한하여 그 기능을 제공합니다.
{% endhint %}

### <mark style="color:$primary;">지원하는 클라이언트 Windows 운영체제</mark>

윈도우 클라이언트를 설치할 수 있는 PC 운영체제(64-bit 권장)의 종류는 아래와 같습니다.

* **Windows** **계열**:

> - Windows XP 32-bit/64-bit
> - Windows Vista 32-bit/64-bit
> - Windows 7 32-bit/64-bit
> - Windows 8 32-bit/64-bit
> - Windows 8.1 32-bit/64-bit
> - Windows 10 32-bit/64-bit
> - Windows 11 32-bit/64-bit
