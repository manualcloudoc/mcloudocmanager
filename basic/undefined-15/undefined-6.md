# 게스트 아이디 관리하기

게스트 아이디는 중앙문서함 내 전용 폴더에서의 업로드/다운로드를 제외한 다른 모든 기능의 사용이 제한되는 특수한 문서중앙화 계정입니다. 사용자는 자신의 개인문서함에서 게스트 아이디를 생성할 수 있으며, 부서문서함의 게스트 아이디는 부서문서함 내 폴더의 폴더관리자가 생성할 수 있습니다.&#x20;

{% hint style="warning" icon="square-poll-horizontal" %}
게스트 아이디에 대한 설명은 **사용자 매뉴얼**의 [**문서 공유 및 협업을 위한 기능 소개**](https://app.gitbook.com/s/vQ0BiQsGY4PT08D7Nfay/basicmodule/undefined-1)를 참고하세요.
{% endhint %}

관리자는 **시스템 설정**에서 개인문서함 및 부서문서함에서의 게스트 아이디의 사용 여부를 설정할 수 있으며, 개별 사용자가 생성할 수 있는 게스트 아이디를 개수를 제한할 수 있습니다.

또한 관리자 웹페이지에서 전체 부서문서함 및 개인문서함의 게스트 현황을 검색하여 조회할 수 있으며, 필요 시 게스트 아이디를 생성하고, 변경, 삭제할 수 있습니다.

#### <mark style="color:$primary;">게스트 아이디 조회하기</mark>

등록된 게스트 아이디를 검색하고 조회하는 방법은 다음과 같습니다.&#x20;

1. **일반 모듈 관리 - Basic - 공유·협업 - 게스트 아이디 - 검색&생성**을 클릭하면 **검색&생성** 페이지가 표시됩니다. **검색&생성** 페이지의 하단에는 현재 등록되어 있는 게스트 아이디 목록이 표시됩니다.

<figure><img src="../../.gitbook/assets/img_001 (7).png" alt=""><figcaption></figcaption></figure>

2. 조회를 원하는 문서함 종류(개인문서함 또는 부서문서함)를 선택하고, 필요 시 검색 조건으로 게스트 ID 또는 게스트 이름을 입력합니다. 이때 검색어의 일부만 입력하여 검색할 수 있습니다.
3. **검색** 버튼을 클릭하면, 검색 조건에 해당하는 게스트의 게스트 ID, 게스트 이름, 생성 위치, 소유자 이름(ID), 등록일, 계정 만료일, 계정 상태가 표시됩니다.

* **생성 위치:** 부서문서함 또는 개인문서함 게스트 여부를 표시합니다.
* **소유자 이름(ID):** 개인문서함 게스트의 경우에는 게스트 아이디를 생성한 사용자명 및 ID가 표시되며, 부서문서함 게스트의 경우에는 게스트 아이디가 생성된 문서함의 이름 및 ID가 표시됩니다.
* **등록일**: 게스트 아이디가 생성된 날짜입니다. 등록일 표시를 클릭하면 등록일 기준 내림차순 또는 오름차순으로 목록을 정렬할 수 있습니다.
* **계정 만료일**: 게스트 아이디 생성 시 설정한 만료일이 표시됩니다.
* **계정 상태** :

> - **사용**:  게스트 아이디의 사용이 허용된 상태입니다.  단, 계정 만료일 이후에는 로그인이 불가능하며, 계정 만료일을 변경한 후 다시 사용할 수 있습니다.
> - **중지**: 관리자 또는 게스트 아이디를 생성한 사용자가 사용 중지로 설정한 상태입니다.

#### <mark style="color:$primary;">게스트 아이디 생성하기</mark>

관리자가 게스트 아이디를 생성하는 방법은 다음과 같습니다.&#x20;

1. **일반 모듈 관리 - Basic - 공유·협업 - 게스트 아이디 - 검색&생성**을 클릭하면 **검색&생성** 페이지가 표시됩니다. 페이지의 하단에는 이미 생성되어 있는 게스트 아이디 목록이 표시됩니다.
2. 게스트 아이디 목록 우측 상단의 **생성** 버튼을 클릭하고, ‘**생성**’창에서 생성할 게스트 아이디의 생성 위치, 게스트 ID, 게스트 이름, 비밀번호, 계정 만료일, 메모 등을 입력합니다.

<div align="left"><img src="../../.gitbook/assets/img_002 (14).png" alt="" width="387"></div>

* **생성 위**치: 개인문서함 또는 부서문서함 중 게스트 아이디를 생성할 위치를 선택합니다. 선택한 문서함 종류에 따라 우측 입력란에 사용자의 이름 또는 ID(개인문서함의 경우), 또는 문서함 이름(부서문서함의 경우)를 입력하고, 드롭다운 목록에서 사용자 또는 문서함을 선택합니다. 이때 문자열의 일부만 입력하여 선택할 수 있습니다.
* **게스트 ID**: 5\~40자의 문자열을 입력합니다.
* **비밀번호**: 8\~20자의 문자열을 입력합니다.
* **계정 만료일**: 생성하는 게스트 아이디의 사용 기한을 제한하지 않거나, 특정 날짜까지 사용 가능하도록 지정할 수 있습니다.

3. **확인**을 클릭하면 입력한 정보에 따라 게스트 아이디가 생성되어 목록에 추가됩니다.

#### <mark style="color:$primary;">게스트 아이디 정보 변경하기</mark>

관리자는 다음과 같이 등록된 게스트 아이디의 계정 정보 및 만료일을 변경하고, 게스트 아이디의 사용을 중지하거나 다시 허용할 수 있습니다.

1. &#x20;**검색&생성** 페이지 하단의 게스트 아이디 목록에서 변경할 게스트 아이디 우측의 **연필** 아이콘을 클릭합니다.
2. ‘**변경**’창에서 **게스트 이름, 비밀번호, 계정 만료일, 메모, 계정 상태**를 변경할 수 있습니다.

* 계정 상태를 중지로 변경하거나 다시 사용으로 변경할 수 있습니다.

3. **확인** 버튼을 클릭하면, 게스트 아이디 정보가 변경됩니다.

<div align="left"><img src="../../.gitbook/assets/img_003 (11).png" alt="" width="386"></div>

#### <mark style="color:$primary;">게스트 아이디 삭제하기</mark>

**검색&생성** 페이지 하단의 게스트 아이디 목록에서 게스트 아이디 항목 우측의 휴지통 아이콘을 클릭하여 해당 게스트 아이디를 등록 삭제할 수 있습니다. \
게스트 아이디를 등록 삭제하더라도 해당 게스트의 전용 폴더 및 저장된 파일은 삭제되지 않으므로, 삭제가 필요한 경우 탐색기에서 직접 삭제해야 합니다.
