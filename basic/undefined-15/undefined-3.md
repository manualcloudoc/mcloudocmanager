# 관리자가 문서링크 복사 사용 이력 확인하는 방법

문서링크 복사 기능의 사용 이력을 기간, 수신자 유형, 승인 처리 상태, 사용자, 승인권자 및 파일 종류별로 검색하여 확인할 수 있습니다.\
1\.    **일반 모듈 관리 - Basic - 공유·협업 - 문서링크 - 이력 보기**를 클릭합니다.

![](<../../.gitbook/assets/img_000 (24).png>)

2\.    **이력 보기** 페이지에서 조회를 원하는 기간, 수신자 유형, 작업 상태를 선택합니다.

1. **수신자 유형**

> 1) 사용자가 문서링크 복사 요청 시 선택한 수신자 유형(**내부/계열사/협력사 직원**)별로 조회가 가능합니다.
> 2) **유형 없음**:  구 버전 문서링크 복사 사용 이력의 경우, 수신자 유형이 **유형 없음**으로 분류됩니다.

1. **작업 상태**

> 1) **대기**: 승인이 필요한 문서링크 복사 요청 후, 승인권자의 승인을 기다리는 상태입니다.
> 2) **승인**: 승인이 필요한 문서링크 복사 요청에 대해 승인권자가 승인을 완료한 상태입니다.
> 3) **반려**: 승인이 필요한 문서링크 복사 요청을 승인권자가 반려한 상태입니다.
> 4) **취소**: 선결재 승인이 필요한 문서링크 복사 요청을 결재 전에 사용자가 취소한 상태입니다.
> 5) **완료**: 승인이 필요 없는 문서링크 복사 요청에 대해 링크 생성이 완료된 상태입니다.

3\.    필요 시 **검색 조건**으로 **생성자 이름/ID, 승인권자 이름/ID, 파일명, 파일 확장자명**을 입력합니다. 이때 검색어의 일부만 입력하여 검색할 수 있습니다. 단, 파일 확장자명으로 검색 시에는 입력한 문자열이 확장자와 일치해야 검색이 가능합니다.

1. **생성자 ID/이름**: 문서링크 복사를 요청한 사용자별로 검색할 수 있습니다.

4    검색 버튼을 클릭하면, 입력한 조건에 해당하는 사용 이력의 **생성 일시, 수신자 유형, 제목, 생성자 이름/ID, 승인권자 이름/ID, 작업 상태**가 표시됩니다.

1. **생성 일시**: 문서링크가 생성된 일시가 표시됩니다.
2. **제목**: 대상 파일의 이름이 표시됩니다. 복수 개의 파일에 대해 문서링크 복사가 요청된 경우, “대표 파일명(외 n개)”와 같이 표시됩니다.

5\.    표시된 이력 목록에서상세보기를 원하는 작업 이력의**제목**을 클릭하면 해당 작업의 상세 정보를 확인할 수 있습니다.

1. **상세보기** 페이지에서 추가적으로 수신자 인증 방식, OTP 인증용 이메일 주소(이메일 OTP 인증 사용 시), 결재 방식 및 승인신청 사유(승인 필요 시), 다운로드 유효기한, 다운로드 가능 횟수, 승인/반려 사유(승인/반려 시)를 확인할 수 있습니다.
2. 하단의 \*\*파일 목록(다운로드 정보)\*\*에 대상 파일 각각의 파일명, 파일 경로, 크기 및 다운로드 횟수(사용자가 설정한 총 다운로드 가능 횟수 대비 수신자가 다운로드한 횟수)가 표시됩니다.
3. 페이지 우측 하단의 목록 버튼을 클릭하면 **이력 보기** 페이지로 돌아갑니다.
