# 관리자가 보안 웹링크 복사 및 보안 링크메일 사용 이력 확인하는 방법

##

&#x20;보안 웹링크 복사 및 보안 링크메일 기능의 사용 이력을 기간, 제목, 승인 처리 상태, 부서, 사용자, 승인권자 및 파일 종류별로 검색하여 확인할 수 있습니다.

1\.    **일반 모듈 관리 - Basic - 공유**·**협업 - 보안-링크메일/웹링크 - 이력 보기**를 클릭합니다.

![](<../../.gitbook/assets/img_000 (25).png>)

2\.   **이력 보기** 페이지에서 조회를 원하는 기간, 링크 종류(**보안웹링크 복사** 또는 **보안링크메일**), 작업 상태를 선택합니다.

1. **작업 상태**

> 1) **대기**: 승인 절차 진행 시, 승인권자의 승인을 기다리는 상태입니다.
> 2) **승인**: 승인 절차 진행 시, 승인권자가 승인을 완료한 상태입니다.
> 3) **반려**: 승인 절차 진행 시, 승인권자가 승인 요청을 반려한 상태입니다.
> 4) **취소**: 선결재 승인 절차 진행 시, 사용자가 결재 전에 요청을 취소한 상태입니다.
> 5) **완료**: 승인이 필요 없는 경우에 링크 생성이 완료된 상태입니다.

3\.    필요 시 **검색 조건**으로 **생성자 이름/ID, 승인권자 이름/ID, 제목, 부서명, 파일명, 파일 확장자명**을 입력합니다. 이때 검색어의 일부만 입력하여 검색할 수 있습니다. 단, 파일 확장자명으로 검색 시에는 입력한 문자열이 확장자와 일치해야 검색이 가능합니다

1. **생성자 이름/ID**: 보안 웹링크 복사 또는 보안 링크메일을 요청한 사용자별로 검색할 수 있습니다.
2. **제목**: 사용자가 보안 웹링크 복사 또는 보안 링크메일 요청 시 입력한 제목으로 검색할 수 있습니다.
3. **부서명**: 사용자의 소속 부서명으로 검색할 수 있습니다.

![Idea](<../../.gitbook/assets/img_001 (24).png>)사용자가 제목 입력 시 입력란에 대상 파일 이름을 포함한 기본 제목이 생성되며, 필요 시 사용자가 수정할 수 있습니다. (예: “문서중앙화 서비스 소개.pdf에 대하여 웹링크 복사/링크메일 보내기를 신청합니다.”)\
4\.    검색 버튼을 클릭하면, 입력한 조건에 해당하는 사용 이력의 **생성 일시, 링크 종류, 제목, 생성자 이름/ID, 부서명, 승인권자 이름/ID, 작업 상태**가 표시됩니다.

1. **생성 일시**: 웹링크가 생성된 일시가 표시됩니다.

> ![Notes](<../../.gitbook/assets/img_002 (20).png>)보안 웹링크 복사 또는 보안 링크메일 요청 즉시 웹링크가 생성되며, 보안 링크메일의 경우, 실제 메일 발송 시점은 설정에 따라 결재 승인 이후가 될 수 있습니다.

5\.    표시된 이력 목록에서상세보기를 원하는 작업 이력의**제목**을 클릭하면 해당 작업의 상세 정보를 확인할 수 있습니다.

1. **상세보기** 페이지에서 추가적으로 내용, 만료일, 수신자 이메일 주소(보안 링크메일 사용 시), 파일목록을 확인할 수 있습니다.

> 1) **내용**:  사용자가 보안 웹링크 복사 또는 보안 링크메일 요청 시 **내용** 란에 입력한 텍스트로, 사용자나 관리자가 요청 내역을 확인하는 용도로 사용됩니다. 수신자에게는 공유되지 않습니다.
> 2) **만료일**: 사용자가 보안 웹링크 복사 또는 보안 링크메일 요청 시 설정한 **사용 만료 일시**(다운로드 유효 기한)가 표시됩니다.
> 3) **파일 목록 (다운로드 수)**: 대상 파일의 이름과 경로, 수신자가 다운로드한 횟수를 확인할 수 있습니다.

1. 페이지 우측 하단의**목록**버튼을 클릭하면**이력 보기**페이지로 돌아갑니다.
