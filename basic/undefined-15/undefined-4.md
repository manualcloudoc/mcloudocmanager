# 관리자가 웹링크 복사 및 링크메일 사용 이력 확인하는 방법

웹링크 복사 및 링크메일 기능의 사용 이력을 기간, 제목, 사용자별로 검색하여 확인할 수 있습니다.​\
1\.    **일반 모듈 관리 - Basic - 공유**·**협업 - 링크메일/웹링크 - 이력 보기**를 클릭합니다.

![](<../../.gitbook/assets/img_000 (26).png>)

2\.    **이력 보기** 페이지에서 조회를 원하는 기간, 링크 종류(**웹링크 복사** 또는 **링크메일**)를 선택하고, 필요 시 다음의 검색 조건을 입력합니다. 이때 검색어의 일부만 입력하여 검색할 수 있습니다.

1. **생성자 이름/ID**: 웹링크 복사 또는 링크메일을 요청한 사용자별로 검색할 수 있습니다.
2. **제목**: 사용자가 웹링크 복사 또는 링크메일 요청 시 입력한 제목으로 검색할 수 있습니다.

> ![Idea](<../../.gitbook/assets/img_001 (25).png>)사용자가 제목 입력 시 입력란에 대상 파일 이름을 포함한 기본 제목이 생성되며, 필요 시 사용자가 수정할 수 있습니다. (예: “문서중앙화 서비스 소개.pdf 외 2개 파일을 보냅니다.”)

3\.    검색 버튼을 클릭하면, 입력한 조건에 해당하는 사용 이력의 **생성 일시, 링크 종류, 제목, 생성자 이름/ID**가 표시됩니다.

1. **생성 일시**: 웹링크가 생성된 일시가 표시됩니다.

4\.    표시된 이력 목록에서상세보기를 원하는 작업 이력의**제목**을 클릭하면 해당 작업의 상세 정보를 확인할 수 있습니다.

1. **상세보기** 페이지에서 추가적으로 내용, 만료일, 수신자 이메일 주소(링크메일 사용 시), 파일목록을 확인할 수 있습니다.

> 1) **내용**:  사용자가 웹링크 복사 또는 링크메일 요청 시 **내용** 란에 입력한 텍스트로, 사용자나 관리자가 요청 내역을 확인하는 용도로 사용됩니다.
> 2) **만료일**: 사용자가 웹링크 복사 또는 링크메일 요청 시 설정한 **사용 만료 일시**(다운로드 유효 기한)가 표시됩니다.
> 3) **파일 목록 (다운로드 수)**: 대상 파일의 이름과 수신자가 다운로드한 횟수를 확인할 수 있습니다. 파일 이름을 클릭하면 해당 파일을 다운로드할 수 있습니다.

1. 페이지 우측 하단의 **목록** 버튼을 클릭하면 **이력 보기** 페이지로 돌아갑니다.
