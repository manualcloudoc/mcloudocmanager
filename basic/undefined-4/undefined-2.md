# 도메인 정보 변경, 삭제하기

이용 중인 도메인의 기본 정보, 용량, 사용자 수를 변경하거나 도메인을 삭제하는 방법을 설명합니다.&#x20;

멀티도메인 기능을 구입한 경우 추가도메인 관리자의 도메인 정보 변경 및 삭제 방법은 기본도메인 관리자와 동일합니다.

### <mark style="color:$primary;">도메인 기본 정보 변경하기</mark>

이용 중인 도메인의 도메인 주소와 사이트명을 변경할 수 있습니다.

1. **일반관리 - 도메인 관리 - 검색**을 클릭합니다.
2. 정보를 변경하려는 도메인을 클릭합니다.
3. **기본 정보** 영역 우측 상단의 **수정** 버튼을 클릭합니다.
4. **‘변경’** 창에서 도메인 주소 또는 사이트명을 수정할 수 있습니다.

* **도메인**: 변경하고자 하는 도메인 주소명을 입력합니다
* **사이트명**: 변경하고자 하는 사이트명을 입력합니다.

5. **확인**을 클릭합니다.

### <mark style="color:$primary;">도메인 삭제하기</mark>

이용 중인 도메인을 삭제할 수 있습니다.

1. **일반관리 - 도메인 관리 - 검색**을 클릭합니다.
2. 삭제하려는 도메인을 클릭합니다.
3. **기본 정보** 영역 우측 상단의**삭제** 버튼을 클릭합니다.
4. **확인**을 클릭합니다.

## 도메인 용량 추가하기

도메인에서 사용하고 있는 전체 용량이 부족한 경우에 추가할 수 있습니다.​

1. **일반관리 - 도메인 관리 - 검색**을 클릭합니다.
2. 용량을 추가할 도메인을 클릭합니다.
3. **용량 상세 목록**에서 **추가**를 클릭합니다.
4. ‘**용량 추가’** 창에서 용량 등의 정보를 입력하여 추가합니다.

* **사용 만료일**:  추가하는 용량은 이 날짜 까지만 사용할 수 있습니다. 이 날짜 이후에는 사용할 수 있는 용량이 이전 상태로 줄어듭니다.

5. &#x20;**확인**을 클릭합니다.

### <mark style="color:$primary;">도메인 용량 변경하기</mark>

도메인의 전체 용량을 변경할 때 사용합니다.

1. **일반관리 - 도메인 관리 - 검색**을 클릭합니다.
2. 용량을 변경할 도메인을 클릭합니다.
3. **용량 상세 목록**에서 **연필** 아이콘을 클릭합니다.
4. ‘**용량 수정’** 창에서 정보를 변경합니다.
5. **확인**을 클릭합니다.

{% hint style="danger" %}
용량 변경 시 이미 사용 중인 전체 용량보다 작게 설정하는 경우 쓰기(업로드, 편집저장)는 차단됩니다. 파일의 열람, 삭제, 이름 변경, 이동은 가능합니다.&#x20;
{% endhint %}

### <mark style="color:$primary;">도메인 용량 삭제하기</mark>

1. **일반관리 - 도메인 관리 - 검색**을 클릭합니다.
2. 용량을 삭제할 도메인을 클릭합니다.
3. **용량 상세 목록**에서 **휴지통**을 클릭합니다.
4. **확인**을 클릭합니다.

### <mark style="color:$primary;">도메인 사용자 수 추가하기</mark>

도메인을 사용할 수 있는 사용자 수를 추가합니다.

1. **일반관리 - 도메인 관리 - 검색**을 클릭합니다.
2. 사용자를 추가할 도메인을 클릭합니다.
3. **사용자 수 상세** 목록에서 **추가**를 클릭합니다.&#x20;
4. ‘**사용자 수 추가'** 창에서 추가할 사용자 수를 입력합니다. 전체 도메인의 사용자 수는 라이선스 내에서만 추가할 수 있습니다.

* **사용 만료일**:  추가하는 사용자수는 이 날짜 까지만 사용할 수 있습니다. 이 날짜 이후에는 사용자수가 이전 상태로 줄어듭니다.

5. **확인**을 클릭합니다.

### <mark style="color:$primary;">도메인 사용자 수 변경하기</mark>

도메인을 사용하는 사용자 수를 변경할 수 있습니다.

1. **일반관리 - 도메인 관리 - 검색**을 클릭합니다.
2. 사용자 수를 변경할 도메인을 클릭합니다.
3. **사용자 수 상세 목록**에서 **연필**아이콘을 클릭합니다.
4. **‘사용자 수 수정’** 창에서 정보를 변경합니다.
5. **확인**을 클릭합니다.

{% hint style="danger" %}
사용자 수 변경 시 이미 생성된 사용자 수보다 적게 설정하는 경우 일부 사용자의 사용이 정지됩니다. 마지막 로그인이 가장 오래된 순서부터 모자란 라이선스 수만큼 정지로 변경됩니다.
{% endhint %}

### <mark style="color:$primary;">도메인 사용자 수 삭제하기</mark>

1. **일반관리 - 도메인 관리 - 검색**을 클릭합니다.
2. 사용자 수를 삭제할 도메인을 클릭합니다.
3. **사용자 수 상세 목록**에서 **휴지통**을 클릭합니다.
4. **확인**을 클릭합니다.
