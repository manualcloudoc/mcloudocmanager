# 보안등급문서 승인권자 지정하기

자료의 반출(문서반출, 외부DRM 반출)이나 링크 공유(문서링크, 보안/대외용 웹링크) 작업의 승인을 위해, 문서함/폴더별로 보안등급문서의 승인권자를 지정하고, 보안등급별 승인 권한을 설정할 수 있습니다.&#x20;

{% hint style="warning" icon="square-poll-horizontal" %}
보안등급문서의 승인권자를 지정하기 위해서는 문서함/폴더별로 승인권자 종류가 지정되어 있어야 합니다.
{% endhint %}

1. 관리자 웹페이지에서 **일반 관리 - 인사 조직 관리 – 승인권자 관리 – 승인권자 지정 – 보안등급문서 승인권자 지정**을 선택합니다.

![](<../../.gitbook/assets/img_001 (37).png>)

2. &#x20;보안등급문서의 승인권자를 지정하고자 하는 문서함/폴더를 선택합니다.



* 문서함/폴더의 승인권자 종류가 **팀 문서관리자**로 지정된 경우

![](<../../.gitbook/assets/img_002 (32).png>)

> * 선택한 문서함/폴더의 보안등급문서 승인권자로 지정할 팀 문서관리자를 조직도에서 선택합니다
> * 보안등급문서 승인 권한 여부를  <img src="../../.gitbook/assets/img_003 (27).png" alt="" data-size="line">(ON)으로 설정합니다.
> * 보안등급별 권한을 설정합니다.



* 문서함/폴더의 승인권자 종류가 **폴더관리자**로 지정된 경우

![](<../../.gitbook/assets/img_004 (20).png>)

* 선택한 문서함/폴더의 폴더관리자 목록이 표시됩니다.
* 보안등급문서의 승인권자로 지정하려는 폴더관리자 우측의 보안등급문서 승인 권한을 <img src="../../.gitbook/assets/img_005 (17).png" alt="" data-size="line"> (ON)으로 설정합니다.
* 보안등급별 권한을 설정합니다.

3\.   **설정** 버튼을 클릭하여 보안등급문서 승인권자 지정을 완료합니다.
