# 보안등급문서 승인권자 지정 이력 보기

문서함/폴더에서 보안등급문서 승인권자를 지정하고, 보안등급별 승인 권한을 설정한 이력을 확인할 수 있습니다.&#x20;

1. 관리자 웹페이지에서 **일반 관리 - 인사 조직 관리 – 승인권자 관리 – 승인권자 지정 – 보안문서 승인권자 지정 이력**을 선택합니다.

<figure><img src="../../.gitbook/assets/img_000 (35).png" alt=""><figcaption></figcaption></figure>

2. **보안등급문서 승인권자 지정 이력** 페이지에서 조회를 위한 조건을 선택합니다.

* **시작일 선택**(필수): 보안등급문서 승인 권한이 설정된 날짜입니다.
* **승인권자 종류**(필수): **폴더 관리자**와 **팀 문서관리자** 중 선택합니다.
* **작업 종류**(필수): 적용, 변경, 해제 중에서 하나 이상 선택합니다.

> - **적용**: 보안등급문서 승인권자로 지정
> - **변경**: 보안등급문서 승인 권한이 변경
> - **해제**: 보안등급문서 승인권자에서 지정 해제

​\
3\. 필요 시 검색 조건으로 검색어를 입력할 수 있으며 검색어의 일부만 입력하여 검색할 수 있습니다.

* 팀 문서관리자 선택 시: 승인권자 이름, 승인권자 ID, **부서명**, 관리자 이름, 관리자 ID
* 폴더관리자 선택 시: 승인권자 이름, 승인권자 ID, **폴더 경로**, 관리자 이름, 관리자 ID



4. **검색** 버튼을 클릭하면, 입력한 조건에 해당하는 보안등급문서 승인권자 지정 이력이 다음과 같이 표시되며 최근에 지정된 작업이 상단에 위치합니다.

* 팀 문서관리자 선택 시

![](<../../.gitbook/assets/img_001 (34).png>)

> * 작업일시, 승인권자 종류, 작업 종류, 승인권자 이름(ID), 부서명, 보안등급문서 승인 권한, 관리자 이름(ID)가 표시됩니다.

* ​폴더관리자 선택 시

![](<../../.gitbook/assets/img_002 (29).png>)

> * 작업일시, 승인권자 종류, 작업 종류, 승인권자 이름(ID), 폴더 경로, 보안등급문서 승인 권한, 관리자 이름(ID)가 표시됩니다.

* 보안등급문서 승인 권한이 있는 경우 **상세보기**를 클릭하면 등급별 승인 권한을 확인할 수 있습니다

<div align="left"><img src="../../.gitbook/assets/img_003 (24).png" alt="" width="443"></div>

* 지정일시를 클릭하면 지정된 날짜의 내림차순 또는 오름차순으로 정렬할 수 있습니다.
