# 자가승인 정책 적용 이력 보기

​자가승인 정책을 부서, 사용자별로 적용한 이력을 다음과 같이 확인할 수 있습니다.&#x20;

1. **일반 관리 - 인사 조직 관리 – 승인권자 관리 – 자가승인 – 정책 적용** 이력을 클릭합니다.

![](<../../.gitbook/assets/img_000 (70).png>)

2. **정책 적용 이력** 페이지에서 조회를 원하는 기간, 대상 구분, 작업 종류를 선택합니다.

* **기간 선택**: 정책이 적용된 기간을 선택합니다.
* **대상 구분**: 정책을 적용한 대상을 사용자와 부서에서 하나 이상 선택합니다.
* **작업 종류**: 조회할 작업 종류를 하나 이상 선택합니다.

> - 적용: 정책 미적용 상태에서 정책을 적용한 경우
> - 변경: 정책이 적용된 상태에서 새로운 정책으로 변경 적용한 경우
> - 해제: 정책을 적용 해제한 경우

3. 필요 시 검색 조건에 사용자 이름/ID, 부서명, 관리자 이름/ID를 입력하여, 검색 결과를 한정할 수 있습니다.  이때 검색어의 일부만 입력하여 검색할 수 있습니다.
4. **검색** 버튼을 클릭하면, 입력한 조건에 해당하는 정책 적용 이력이 하단에 표시됩니다.

<figure><img src="../../.gitbook/assets/img_001 (65).png" alt=""><figcaption></figcaption></figure>

* 적용일시: 정책 적용 작업을 수행한 일시
* 대상 구분: 정책이 적용된 단위를 사용자, 부서로 구분
* 사용자 이름(ID)/부서명: 정책 적용 작업의 대상인 사용자 또는 부서 정보
* 작업 종류: 적용, 변경, 해제 중 수행된 작업
* 적용 정책: 적용, 변경, 해제된 정책명
* 관리자 이름(ID): 정책 적용 작업을 수행한 관리자 정보
* 임시 정책 여부(허용 기간): 임시 정책으로 설정한 경우 허용 기간, 임시 정책이 아닌 경우 표시되지 않음
