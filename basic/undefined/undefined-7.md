# 서비스관리자 보안등급 설정하기

정보보호관리자는 서비스관리자의 보안등급을 설정하여 서비스관리자의 권한을 제한할 수 있습니다.

1. **서비스관리자 관리 - 보안등급 설정**을 클릭합니다. 상단에서 현재 설정된 보안등급을 확인할 수 있습니다.
2. 상단에서 현재 설정된 보안등급을 확인할 수 있습니다. 보안등급은 상, 중, 하, 커스텀으로 설정할 수 있습니다. 보안등급 ‘하’는 모든 권한을, ‘상’은 가장 제한된 권한을 허용합니다. ‘커스텀’ 보안등급은 개별 항목별로 설정합니다.
3. 보안등급 상, 중, 하에서 선택할 시 원하는 보안등급의 **설정**을 클릭합니다.
4. 커스텀의 경우 허용하고자 하는 항목의 **토글** 버튼 ![](<../../.gitbook/assets/img_000 (58).png>)을 클릭하여 활성화를 시킨 후 **설정**을 클릭합니다.

![](<../../.gitbook/assets/img_001 (54).png>)

{% hint style="warning" icon="square-poll-horizontal" %}
파일 목록과 권한 설정에 대한 권한은 서비스관리자가 기본으로 갖는 권한입니다. 따라서, 커스텀 보안등급의 경우에도 X로 변경할 수 없습니다.

* **파일 목록**: 폴더 내 파일 목록을 볼 수 있는 권한
* **권한 설정**: 폴더 관리자로서 설정할 수 있는 권한
{% endhint %}
