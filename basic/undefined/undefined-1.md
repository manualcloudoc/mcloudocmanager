# 관리자 계정의 이름, 비밀번호 변경하기

#### <mark style="color:$primary;">정보보호관리자 계정의 이름, 비밀번호 변경하기</mark>

정보보호관리자 계정의 이름, 비밀번호는 다음과 같은 방법으로 변경할 수 있습니다.

1. 웹페이지 우측 상단의 **정보보호관리자 이름**을 클릭합니다.
2. **계정관리**메뉴를 선택합니다.
3. 변경할 이름과 비밀번호를 입력합니다.
4. **수정**을 클릭합니다.

#### <mark style="color:$primary;">서비스관리자 계정의 이름, 비밀번호 변경하기</mark>

서비스관리자 계정의 이름, 비밀번호를 다음과 같이 변경할 수 있습니다.

1. 웹페이지 우측 상단의**서비스관리자 이름**을 클릭합니다.
2. **계정관리**메뉴를 선택합니다.
3. 변경할 이름과 비밀번호를 입력합니다.
4. **수정**을 클릭합니다.
