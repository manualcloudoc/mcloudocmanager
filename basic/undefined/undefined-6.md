# 서비스관리자 계정 변경, 삭제하기

정보보호관리자는 서비스관리자의 계정정보를 변경하거나 서비스관리자 계정을 삭제할 수 있습니다.

#### <mark style="color:$primary;">서비스관리자 계정 정보 변경하기</mark>

1. **서비스관리자 관리 - 계정 설정**을 클릭합니다.
2. 서비스관리자의 **연필** 아이콘을 클릭합니다.

<figure><img src="../../.gitbook/assets/img_000 (55).png" alt=""><figcaption></figcaption></figure>

3. 변경할 정보(이름, 비밀번호, 메뉴 선택)를 입력 후 수정을 클릭합니다.&#x20;

![](<../../.gitbook/assets/img_001 (52).png>)

{% hint style="warning" icon="square-poll-horizontal" %}
서비스관리자 ID는 변경할 수 없습니다.
{% endhint %}

#### <mark style="color:$primary;">서비스관리자 계정 삭제하기</mark>

1. **서비스관리자 관리 - 계정 설정**을 클릭합니다.
2. 삭제할 서비스관리자의 **휴지통**을 클릭합니다.
3. ‘**삭제**’창에서 **확인**을 클릭합니다.
