# 정보보호관리자 메뉴 편집하기

정보보호관리자는 사용하지 않는 메뉴를 제거하여 화면에서 보이지 않게 할 수 있습니다.​

1. 웹페이지 우측 상단의 **정보보호관리자 이름**을 클릭합니다.
2. **계정관리**를 클릭합니다.
3. **메뉴 선택**을 클릭하여 사용할 메뉴만 체크합니다.
4. **확인**을 클릭합니다.
