# 관리자 계정 이미지 등록하기

정보보호관리자의 프로필 이미지는 다음과 같이 등록할 수 있습니다.

1. 웹페이지 우측 상단의 **정보보호관리자 이름**을 클릭합니다.
2. **계정관리** 메뉴를 선택합니다.
3. **카메라** 아이콘을 선택하여 프로필 이미지를 업로드합니다. 권장하는 사진의 크기는 가로 256px, 세로 256px이며, 파일 사이즈는 5MB 이하로, 이미지 파일 종류는 png, jpg, gif 만 가능합니다.
4. **수정**을 클릭합니다.
