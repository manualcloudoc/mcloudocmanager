# 서비스관리자 계정 추가, 메뉴 설정하기

### <mark style="color:$primary;">서비스관리자 계정 추가하기</mark>

정보보호관리자는 서비스관리자에게 권한을 위임할 수 있고, 여러 명의 서비스관리자에게 담당 업무에 따라 사용 가능한 메뉴를 지정할 수 있습니다. 서비스관리자에게 업무를 지정하기 위해서는 먼저 계정을 생성해야 합니다.

1. **서비스관리자 관리 - 계정 설정**을 클릭합니다.
2. **추가**를 클릭합니다.
3. **서비스관리자 추가** 페이지에서 이름, ID, 이메일, 비밀번호를 작성합니다.

* **이름(필수)**: 영문, 한글, 숫자, 일부 특수문자를 사용할 수 있습니다. 특수문자(/ ? , ; | \` + < > # $ % & \ ‘ “)는 사용할 수 없습니다.
* **ID(필수)**: 5자 이상 40자 이하의 영문 소문자, 숫자, 일부 특수문자(- @^$()\_-=\[]\~!.{})를 사용할 수 있습니다.  동일한 ID가 있는 경우 등록되지 않으며 서비스관리자 생성이 완료되면 ID를 변경할 수 없습니다.
* **비밀번호(필수)**: 별도의 설정이 없는 경우8자 이상 20자 이하로 영문, 숫자, 특수문자를 혼합하여 설정할 수 있습니다.

![](<../../.gitbook/assets/img_000 (56).png>)

4. **메뉴 선택** 버튼을 클릭하여 ‘**메뉴 선택**’ 창에서 서비스관리자에게 허용할 메뉴만 체크합니다.

<div align="left"><figure><img src="../../.gitbook/assets/img_001 (53).png" alt="" width="563"><figcaption></figcaption></figure></div>

5. **확인**을 클릭합니다.

### <mark style="color:$primary;">서비스관리자 메뉴 설정하기</mark>

정보보호관리자는 서비스관리자에게 허용할 메뉴를 변경할 수 있습니다.

1. **서비스관리자 관리 - 계정 설정**을 클릭합니다.
2. 메뉴를 변경할 서비스관리자의 **연필** 아이콘을 클릭합니다.
3. **메뉴 선택**을 클릭하여 허용할 메뉴만 체크합니다.
4. **확인**을 클릭합니다.
5. **수정**을 클릭합니다.
