# 관리자가 모바일 기기 등록 및 관리하는 방법

모바일 앱으로 문서중앙화 서비스에 로그인하기 위해서는 모바일 기기가 등록되어 있어야 합니다. 등록되지 않은 모바일 기기에서 처음 로그인 시도 시, 시스템 설정에 따라 기기가 자동으로 등록되거나 사용자가 등록 신청을 해야 합니다. 등록을 신청한 경우에는 관리자의 승인 후에 기기가 등록됩니다.

## 등록된 모바일 기기 관리하기

관리자는 등록된 모바일 기기를 조회하거나 삭제하고, 모바일 기기의 상태 정보 및 2단계 인증 승인용 기기 설정을 변경할 수 있습니다.

### 모바일 기기 등록 정보 조회하기

1\.    **일반 모듈 관리 - Basic - 로그인 기기별 제어 - 모바일 - 디바이스 관리**를 클릭합니다.

![](<../../.gitbook/assets/img_000 (23).png>)

2\.    등록되어 있는 모든 모바일 기기의 등록 정보를 확인할 수 있습니다. 우측 상단에서**사용자ID/이름**을 입력하여 검색할 수도 있습니다.

1. **상태**: **정상** 또는 **분실신고** 여부를 표시합니다.

> 1) **정상**: 모바일 기기의 서버 접속이 허용되어 정상적으로 사용 가능한 상태
> 2) **분실신고**: 모바일 기기의 서버 접속을 차단한 상태

1. **2단계 인증 승인용 기기**: 해당 모바일 기기가 **2단계 인증 승인용 기기**로 지정된 경우, **지정**으로 표시됩니다.

### 모바일 기기의 상태 정보 변경하기

모바일 기기의 서버 접속을 허용 또는 차단하기 위해 모바일 기기의 상태 정보를 변경할 수 있습니다.

1\.    등록되어 있는 모바일 기기 목록에서 상태를 변경할 기기 우측의 **연필**아이콘을 클릭합니다.\
2\.    **‘수정’창에서 모바일 기기의 상태를 정상또는분실신고**로 변경 선택할 수 있습니다. **분실신고**상태로 설정 후사용자가 로그인 시도 시, 모바일 기기의 반출 보안디스크에 저장된 모든 자료를 삭제하고 서버 접속을 차단합니다. **정상**상태로 설정하면 모바일 기기를 다시 정상적으로 사용할 수 있습니다.\
![](<../../.gitbook/assets/img_001 (22).png>)\
3\.    **확인**을 클릭합니다.

### 2단계 인증 승인용 기기 변경하기

모바일 기기를 **2단계 인증 승인용 기기**로 지정하거나, 이미 지정된 경우 다른 기기로 변경할 수 있습니다. 기기 지정 또는 변경 시 **사용자 계정 관리** 메뉴의 **모바일 기기 관리** 페이지가 연결됩니다.\
1\.    등록되어 있는 모바일 기기 목록에서 변경할 기기 우측의 **연필**아이콘을 클릭합니다.

2\.    \*\*‘수정’\*\*창에서 모바일 기기의 **2단계 인증 승인용 기기** 정보를 변경할 수 있습니다.

1. 기기가 2단계 인증 승인용 기기로 지정되어 있지 않은 경우: **기기 지정** 링크를 클릭합니다.
2. 지정되어 있으나 기기를 변경하는 경우: **지정**(**다른 기기로 변경)** 링크를 클릭합니다.

![](<../../.gitbook/assets/img_002 (19).png>)

3\.    **사용자 계정 정보**의 **모바일 기기 관리**페이지로 이동하여, **2단계 인증 승인용기기**를 지정하거나 변경합니다. 사용자에게 등록된 모바일 기기 중 한 대만 2단계 인증 승인용 기기로 지정할 수 있습니다.

![Notes](<../../.gitbook/assets/img_003 (17).png>)**2단계 인증 승인용 기기**를 지정하는 방법은 아티클 **사용자 계정 관리에서 모바일 기기 관리하기** 내 목차\*\*[2단계 인증 승인용 기기를 변경하는 방법](../undefined-5/undefined-2.md)\*\*을 참고하세요.&#x20;

### 모바일 기기 삭제하기

모바일 기기는 서버에 설정된 최대 대수만큼만 등록이 가능하므로, 불필요한 기기는 사전에 삭제하는 것이 좋습니다. 사용자가 모바일 기기를 변경하여 새로운 기기를 등록한 경우, 더 이상 사용하지 않을 기존 기기는 관리자 또는 사용자가 직접 삭제해야 합니다.\
1\.    등록되어 있는 모바일 기기 목록에서 삭제할 기기 우측의 **휴지통** 아이콘을 클릭합니다.

2\.    **‘삭제’** 창에서 **확인**을 클릭합니다.

## **사용자의 모바일 기기 등록 신청에 대하여 승인/반려하는 방법**

업무 또는 인증용으로 모바일 기기를 사용하기 위해서는 모바일 기기가 등록되어 있어야 합니다. 관리자는 사용자의 모바일 기기 등록 신청을 다음과 같이 승인 또는 반려할 수 있습니다.

1\.    **일반 모듈 관리 - Basic - 로그인 기기별 제어 - 모바일 - 요청 처리**를 클릭합니다.

![](<../../.gitbook/assets/img_004 (12).png>)

2\.    목록에서 처리할 요청 건을 체크하여 선택한 후 **승인** 또는 **반려** 버튼을 클릭합니다.

3\.    **확인**을 클릭합니다.&#x20;

## 사용자의 모바일 기기 등록 관리 이력보기

등록 신청한 모바일 기기의 승인/반려 이력을 확인할 수 있습니다.&#x20;

1\.    **일반 모듈 관리 - Basic - 로그인 기기별 제어 - 모바일 - 요청 처리 이력**을 클릭합니다.

![](<../../.gitbook/assets/img_005 (11).png>)

2\.    관리자가 처리한 **작업 기간**(필수)과 **작업 종류**(필수)를 선택한 후, 필요 시 **검색 조건**에 **사용자 이름** 또는 **ID**를 입력합니다. \
3\.    **검색**을 클릭합니다.\
4\.    선택한 조건에 해당하는 모바일 기기 등록 요청 건의 **작업일**, **사용자 이름/ID**, **기종**, **버전**, **디바이스 ID**, **작업 종류**를 확인할 수 있습니다.
