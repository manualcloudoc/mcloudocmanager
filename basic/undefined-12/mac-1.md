# Mac 기기 등록 및 관리하기

MacOS 기기에서 문서중앙화 서비스에 로그인하기 위해서는 기기가 등록되어 있어야 합니다. 관리자는 사용자의 Mac 기기 등록 요청을 승인 또는 반려할 수 있으며, 등록된 Mac 기기를 조회, 삭제하거나 기기의 상태 정보를 변경할 수 있습니다.&#x20;

### <mark style="color:$primary;">등록된 Mac 기기 관리하기</mark>

#### Mac 기기 등록 정보 조회하기

1. **일반 모듈 관리 - Basic - 로그인 기기별 제어 - Mac 기기 - 디바이스 관리**를 클릭합니다.

<figure><img src="../../.gitbook/assets/img_000 (10).png" alt=""><figcaption></figcaption></figure>

2. 등록되어 있는 모든 MacOS기기의 등록 정보를 확인할 수 있습니다. 우측 상단에서 **사용자ID/이름**을 입력하여 검색할 수도 있습니다.

* **디바이스 ID**: Mac 기기의 디바이스 ID
* **버전:** Mac기기의 소프트웨어 버전
* **상태**: **정상** 또는 **분실신고** 여부

> - **정상**: Mac 기기의 서버 접속이 허용되어 정상적으로 사용 가능한 상태
> - **​분실신고:** Mac 기기의 서버 접속을 차단한 상태

#### 등록된 Mac 기기의 상태 정보 변경하기

Mac 기기의 서버 접속을 허용 또는 차단하기 위해 Mac 기기의 상태 정보를 변경할 수 있습니다.

1. 등록되어 있는 Mac 기기 목록에서 상태를 변경할 기기 우측의 **연필** 아이콘을 클릭합니다.
2. **‘수정’** 창에서 Mac 기기의 **상태**를 **정상** 또는 **분실신고**로 변경 선택할 수 있습니다.

<div align="left"><figure><img src="../../.gitbook/assets/img_001 (18).png" alt="" width="507"><figcaption></figcaption></figure></div>

3. **확인**을 클릭합니다.

#### 등록된 Mac 기기 삭제하기

Mac 기기는 서버에 설정된 최대 대수만큼만 등록이 가능하므로, 불필요한 기기는 사전에 삭제하는 것이 좋습니다. 사용자가 Mac 기기를 변경하여 새로운 기기를 등록한 경우, 더 이상 사용하지 않을 기존 기기는 관리자 또는 사용자가 직접 삭제해야 합니다.

1. 등록되어 있는 Mac 기기 목록에서 삭제할 기기 우측의 **휴지통** 아이콘을 클릭합니다.
2. **‘삭제’**&#xCC3D;에서 **확인**을 클릭합니다.

### <mark style="color:$primary;">사용자의 Mac 기기 등록 신청에 대하여 승인/반려하는 방법</mark>

관리자는 사용자의 Mac 기기 등록 신청을 다음과 같이 승인 또는 반려할 수 있습니다.&#x20;

1. **일반 모듈 관리 - Basic - 로그인 기기별 제어 - Mac기기- 요청 처리**를 클릭합니다.

<figure><img src="../../.gitbook/assets/img_002 (13).png" alt=""><figcaption></figcaption></figure>

2. 목록에서 처리할 요청 건을 체크하여 선택한 후 **승인** 또는 **반려** 버튼을 클릭합니다.
3. **확인**을 클릭합니다. &#x20;

### <mark style="color:$primary;">사용자의 Mac 기기 등록 관리 이력보기</mark>

Mac기기 등록 요청에 대한 승인/반려 이력을 확인할 수 있습니다.​

1. **일반 모듈 관리 - Basic - 로그인 기기별 제어 - Mac 기기- 요청 처리 이력**을 클릭합니다.

![](<../../.gitbook/assets/img_003 (10).png>)

2. 관리자가 처리한 **작업 기간**(필수)과 **작업 종류**(필수)를 선택한 후, 필요 시 **검색 조**건에 **사용자 이름** 또는 **ID**를 입력합니다.&#x20;
3. **검색**을 클릭합니다.
4. 선택한 조건에 해당하는 Mac 기기 등록 요청 건의 **작업일, 사용자 이름/ID, 버전, 디바이스ID, 작업 종류**를 확인할 수 있습니다.
