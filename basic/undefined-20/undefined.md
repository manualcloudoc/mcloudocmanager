# 자산화 폴더 설정하기

부서문서함의 특정 폴더를 **자산화 폴더**로 설정하면 폴더 내 자료를 회사의 자산으로 그대로 보존할 수 있습니다. 자산화 폴더는 파일의 편집/수정이 제한되는 폴더로 자산화 폴더의 설정/해제는 서비스관리자만 가능합니다.자산화 폴더의 특징은 다음과 같습니다.

1. 자산화 폴더 내의 파일은 읽기 전용으로 열리며 편집과 수정을 할 수 없습니다
2. 자산화 폴더 내 파일/폴더의 복사는 가능하나 이동은 할 수 없습니다.
3. 자산화 폴더의 하위 폴더는 자산화 폴더로 설정/해제할 수 없습니다.
4. 자산화 폴더의 하위 폴더는 자산화 폴더가 됩니다.
5. 자산화 폴더와 하위 폴더의 권한 설정은 일반 폴더와 동일하게 서비스관리자, 폴더관리자가 할 수 있습니다.

일반 폴더를 자산화 폴더로 설정하거나, 자산화 폴더 설정을 해제하는 방법은 다음과 같습니다.\
1\.    문서중앙화 웹페이지에 서비스관리자로 로그인하여 에이전트를 실행합니다.![](<../../.gitbook/assets/img_000 (73).png>)

2\.    윈도우 탐색기에서 자산화 자산화 폴더로 지정하려는 폴더를 선택하고 마우스를 우클릭하여 **폴더 관리 – 자산화 설정/해제**를 선택합니다. \
![](<../../.gitbook/assets/img_001 (68).png>)

3\.    자산화 폴더가 아닌 경우에는 자산화 폴더로 설정되고 폴더 아이콘이   ![](<../../.gitbook/assets/img_002 (57).png>) 로 변경됩니다.  자산화 폴더로 이미 설정되어 있는 경우에는 자산화 폴더에서 해제되고 일반 폴더 아이콘 ![](<../../.gitbook/assets/img_003 (46).png>)이 표시됩니다.
