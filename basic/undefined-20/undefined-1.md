# 자산화 폴더 분류 설정하기

서비스관리자는 부서문서함의 일부 폴더를 쓰기/편집을 제한하는 자산화 폴더로 설정할 수 있습니다. 자산화 폴더는 카테고리별로 분류하여 관리할 수 있으며, 이를 위해 관리자(서비스관리자 또는 정보보호관리자)가 자산화 폴더의 분류(카테고리)를 생성/변경/삭제할 수 있습니다.

![Notes](<../../.gitbook/assets/img_000 (72).png>)자산화 폴더에 카테고리(분류)를 설정하는 작업은 **자산화 폴더 – 자산화 폴더 설정 현황**에서 수행합니다. 자세한 내용은 \*\*[자산화 폴더 설정 현황 보기](undefined-2.md)\*\*를 참고하세요.

### 자산화 폴더 분류 생성하기

자산화 폴더의 분류를 생성하는 방법은 다음과 같습니다.

1\.    **일반 모듈 관리 - Basic - 자산화 폴더 – 자산화 폴더 분류 설정**을 선택하면 현재 생성되어 있는 분류 현황이 표시됩니다.\
​\
![](<../../.gitbook/assets/img_001 (67).png>)\
2\.    자산화 폴더 분류를 생성하고자 하는 상위의 분류를 선택하고  ![](<../../.gitbook/assets/img_002 (56).png>)를 클릭합니다.

3\.    오른쪽 입력 화면 상단에 있는 **\*분류순서**에는 생성하는 분류의 상위 분류가 표시됩니다. **분류명**과 **설명**을 입력한 후 **추가**를 클릭합니다.\
![](<../../.gitbook/assets/img_003 (45).png>)\
&#x20;4\.    추가를 확인하는 알림 창에서 **확인**을 클릭하면 새로운 자산화 폴더 분류가 생성됩니다.

### 자산화 폴더 분류 변경하기

자산화 폴더 분류의 분류명 또는 설명을 변경하는 방법은 다음과 같습니다.&#x20;

1\.    **일반 모듈 관리 - Basic - 자산화 폴더 – 자산화 폴더 분류 설정**을 선택합니다.\
​\
2\.    좌측의 분류 트리에서 변경하려는 분류의 상위 분류를 선택하면 우측에 해당 분류의 하위 분류들이 나열됩니다.

3\.    변경하고자 하는 항목의 분류명을 체크하고 우측 상단의 변경을 클릭합니다.

![](<../../.gitbook/assets/img_004 (35).png>) \
4\.   변경 내용을 입력하고 **변경** 버튼을 클릭합니다.\
![](<../../.gitbook/assets/img_005 (30).png>) \
5\.   변경을 확인하는 알림 창에서 **확인**을 클릭하면 변경된 내용이 저장됩니다.

### 자산화 폴더 분류 삭제하기

자산화 폴더 분류를 삭제하는 방법은 다음과 같습니다. \
​\
1\.    **일반 모듈 관리 - Basic - 자산화 폴더 – 자산화 폴더 분류 설정**을 선택합니다.\
​\
2\.    좌측의 분류 트리에서 삭제하려는 분류의 상위 분류를 선택하면 해당 분류의 하위 분류들이 나열됩니다.

3\.    항목을 하나 이상 선택하여(좌측의 체크 박스에 표시) 우측 상단의 **삭제** 버튼을 클릭합니다.

![](<../../.gitbook/assets/img_006 (24).png>) \
4\.    삭제 확인 알림 창에서 **확인**을 클릭하여 삭제합니다.
