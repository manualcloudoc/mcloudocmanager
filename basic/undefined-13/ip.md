# IP 대역별로 사용자의 문서함 작업 허용/차단하기

IP 대역별로 사용자의 문서함 작업(파일 목록 보기, 업로드, 다운로드)을 허용/차단할 수 있습니다.

#### <mark style="color:$primary;">IP 필터링 추가하기</mark>

1. **일반 모듈 관리 - Basic - 문서함 관리 - I/O 제어 - IP필터링**을 클릭합니다.&#x20;

<figure><img src="../../.gitbook/assets/img_000 (8).png" alt=""><figcaption></figcaption></figure>

2. &#x20;좌측 상단에서 **등록하지 않은** **IP** 의 문서함 작업을 허용/차단할지 여부를 선택합니다.

* **모두 허용**: 등록하지 않은 IP에 대해서 모두 허용합니다.
* **모두 차단**: 등록하지 않은 IP에 대해서 모두 차단합니다.

3. **추가** 버튼을 클릭합니다.
4. **'IP 추가'** 창에서 시작 IP, 끝 IP 정보를 입력 후 각 작업의 허용/차단 여부를 선택합니다.

* **시작 IP, 끝 IP**: 입력 형식 예) 255.255.0.0
* **작업**: 파일 목록 보기, 업로드, 다운로드 작업의 허용/차단 설정

<div align="left"><figure><img src="../../.gitbook/assets/img_001 (13).png" alt="" width="312"><figcaption></figcaption></figure></div>

5. **확인**을 클릭합니다.

#### <mark style="color:$primary;">IP 대역의 문서함 허용 작업 변경하기</mark>

1. IP대역 목록에서 변경할 IP 대역의 **연필** 아이콘을 클릭합니다.
2. **‘변경’** 창에서 작업 정보를 변경합니다.

* **작업**: 파일 목록 보기, 업로드, 다운로드 작업의 허용/차단 설정

3. **확인**을 클릭합니다.

#### <mark style="color:$primary;">IP 필터링 삭제하기</mark>

1. IP 대역 목록에서 삭제할 IP 대역의 **휴지통**을 클릭합니다.
2. **‘삭제’** 창에서 확인을 클릭합니다.
