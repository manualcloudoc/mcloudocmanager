# 관리자가 웹에서 폴더 및 파일 정보 조회하는 방법

관리자는 문서함 내 폴더 및 파일 현황을 웹에서 확인할 수 있습니다. 확인하는 방법은 다음과 같이 2가지가 있습니다.

1. 문서함 내 파일 목록의 파일명, 수정일시, 파일 크기 정보를 **실시간으로 확인**하는 방법
2. 배치 작업으로 **DB에 수집**해 놓은 파일 정보를 **빠르게 조회**하는 방법

## 관리자가 웹에서 실시간 파일 현황 보는 방법

부서문서함 내 폴더 및 파일 목록과 파일 정보를 실시간으로 확인할 수 있습니다.\
1\.    **일반 모듈 관리 - Basic - 문서함 관리 - 폴더 파일 관리 - 파일 현황**을 클릭합니다.

![](<../../.gitbook/assets/img_000 (27).png>)

2\.    좌측 상단에서 **부서문서함**을 선택합니다.

3\.    **폴더 목록**에서 파일 정보를 확인할 **폴더명**을 클릭합니다.

4\.    우측의 **파일 목록** 영역에서 선택한 폴더에 존재하는 파일의 이름, 수정일시, 파일 크기를 확인합니다.

## 관리자가 수집된 파일 정보 조회하는 방법

문서함내 전체 파일 정보를 배치 작업을 통해 일정 기간 내 최신 정보로 DB 화하여 제공합니다.  \
실시간 정보는 아니지만 서버에 존재하는 파일이 방대한 경우 문서함내 파일 정보를 빠르게 조회 및 다운로드할 수 있습니다.

### 개인문서함/부서문서함 내 수집된 파일 정보 조회하기

개인문서함과 부서문서함 내 파일에 대해 수집된 파일 정보를 조회할 수 있습니다.

1\.    **일반 모듈 관리 - Basic - 문서함 관리 - 폴더 파일 관리 - 파일 조회**를 클릭합니다.

![](<../../.gitbook/assets/img_001 (26).png>)\
&#x20;2\.    검색 조건을 입력합니다.

1. **문서함**: 모든 개인문서함과 부서문서함 중에서 선택할 수 있습니다.
2. **폴더 선택**

\-        **전체**: 선택한 문서함 내의 모든 파일을 조회합니다.

\-          **폴더 선택**: \*\*‘\*\***폴더 선택’** 창에서 조회할 폴더만 체크하여 선택한 후 **확인**을 클릭합니다.

![](<../../.gitbook/assets/img_002 (22).png>)

\-          선택한 폴더를 해제하려면 폴더명 왼쪽 ![](<../../.gitbook/assets/img_003 (19).png>) 아이콘을 클릭합니다.   &#x20;

> <img src="../../.gitbook/assets/img_004 (13).png" alt="" data-size="original">

1. **수정 기간 선택**(필수)**:** 파일 수정일을 기준으로 조회합니다.
2. **문서보안등급:** 문서보안등급을 선택하여 설정할 수 있습니다.

&#x20;         \- **등급 없음** : 보안등급이 지정되지 않은 문서

1. **수정자 이름/ID/부서명, 파일명**으로 검색할 수 있습니다.

3\.    **검색**을 클릭합니다.

4\.    목록에서는 파일이 조회된 건수, 문서함, 폴더명, 파일명, 문서보안등급, 수정자 이름/ID, 수정자부서명, 수정일시를 확인할 수 있습니다.

![](<../../.gitbook/assets/img_005 (12).png>)

### 문서함 내 파일 정보 엑셀 파일로 추출하기

**파일 조회** 메뉴에서 검색한 파일 목록을 엑셀 파일로 다운로드할 수 있습니다.

1\.    **일반 모듈 관리 - Basic - 문서함 관리 - 폴더 파일 관리 - 파일 조회**를클릭합니다.

2\.    조건을 입력하여 엑셀 파일로 추출할 파일만 **검색**합니다.\
3\.    파일 목록 우측 상단의   ![](<../../.gitbook/assets/img_006 (11).png>)를 클릭합니다.

4\.    **‘엑셀파일로 저장하기’** 창에서 저장할 범위를 선택하고 **다운로드** 버튼을 클릭합니다.  \
&#x20;\
&#x20;           ![](<../../.gitbook/assets/img_007 (10).png>)

5\.   엑셀 파일은 폴더별로 생성되고 하나의 Zip 파일에 묶여서 다운로드됩니다. 압축파일을 풀어서 엑셀 파일을 확인합니다.

![](<../../.gitbook/assets/img_008 (9).png>)

1. Zip 파일명 예: 문서 현황.zip
2. 엑셀 파일명 규칙: 선택한 **폴더명**\*\*\_\*\*\*\*엑셀파일생성일자\*\*.xlsx
3. 엑셀 파일은 문서함의 1 depth 폴더 기준으로 생성됩니다. 하위 폴더인 2depth, 3depth 등에 위치한 파일의 정보는 상위 폴더인 1depth 폴더명의 엑셀 파일에 포함됩니다.
