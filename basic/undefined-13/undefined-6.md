# 부서문서함 내 폴더를 드라이브로 연결하기

매번 부서문서함에서 특정 폴더를 찾을 필요 없이 해당 폴더를 별도의 드라이브로 연결하여 사용할 수 있습니다. 폴더 드라이브 연결 방법은 다음과 같습니다.\
&#x20;

#### <mark style="color:$primary;">폴더 드라이브 연결 설정하기</mark>

1. **일반 모듈 관리 - Basic - 문서함 관리 - 폴더 드라이브 연결 - 연결 관리**를 클릭합니다.
2. 좌측 상단에서 **문서함**을 선택합니다.
3. **폴더 목록**에서 드라이브로 연결하여 사용할 폴더를 선택합니다.&#x20;

<figure><img src="../../.gitbook/assets/img_000 (42).png" alt=""><figcaption></figcaption></figure>

4. 우측 영역의 **드라이브명** 항목에 사용할 드라이브 이름을 입력합니다.
5. **추가** 버튼을 클릭합니다.
6. **‘추가’**&#xCC3D;에서 폴더 드라이브의 연결 대상인 부서 또는 사용자를 등록합니다.

<div align="left"><img src="../../.gitbook/assets/img_001 (41).png" alt="" width="467"></div>

등록 방법은 다음과 같습니다.

&#x20;<sup><sub>①<sub></sup> **부서/사용자** 목록에서 부서 또는 사용자를 선택합니다. 또는 검색 입력란에 부서명, 사용자 이름, 사용자 ID를 검색하여 선택할 수도 있습니다.

&#x20;<sup><sub>②<sub></sup> 오른쪽 화살표를 클릭합니다. **선택된 부서 및 사용자** 목록에서 추가한 대상을 확인할 수 있습니다.

&#x20;<sup><sub>③<sub></sup> **확인**을 클릭합니다.

7. 드라이브 연결 시 사용할 드라이브 문자를 설정합니다. **드라이브 문자 설정** 항목에서 A - Z까지의 문자 또는 자동 선택으로 설정합니다. 부서 또는 사용자별로 값을 다르게 설정할 수 있습니다.
8. **적용**버튼을 클릭합니다.<br>

예) 부서문서함 내 ‘해외마케팅 신규자료’ 폴더를 드라이브(I:)로 연결

#### <mark style="color:$primary;">폴더 드라이브 연결 해제하기</mark>

폴더의 드라이브 연결을 다음과 같은 방법으로 해제할 수 있습니다.

1. **일반 모듈 관리 - Basic - 문서함 관리 - 폴더 드라이브 연결 – 연결 관리**를 클릭합니다.
2. 좌측 상단에서 **문서함**을 선택합니다.
3. **폴더 목록**에서 드라이브 연결을 해제할 폴더를 클릭합니다.![Idea](<../../.gitbook/assets/img_004 (24).png>)드라이브 연결 설정이 되어 있는 폴더는 <img src="../../.gitbook/assets/img_005 (20).png" alt="" data-size="line">으로 표시합니다.
4. **폴더 드라이브 연결 대상** 목록에서 삭제할 부서 또는 사용자의 **휴지통**을 클릭합니다.
5. **적용** 버튼을 클릭합니다.
