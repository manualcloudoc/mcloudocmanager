# 폐쇄 부서문서함 관리하기

부서문서함을 생성했던 부서가 해당 부서문서함을 삭제하지 않은 상태에서 폐쇄(조직도에서 삭제)되는 경우, 해당 문서함은 폐쇄 부서문서함으로 관리됩니다. 관리자는 다음과 같이 폐쇄 부서문서함 목록을 확인하고, 필요 시 삭제할 수 있습니다.

1. 관리자 웹페이지 좌측 메뉴에서 **일반 모듈 관리 – Basic – 문서함 관리 – 폐쇄 부서문서함 관리**를 클릭합니다.
2. 폐쇄 부서문서함 관리 페이지에서 폐쇄 부서문서함 목록을 확인할 수 있습니다. 목록에는 폐쇄 부서문서함의 이름, 용량, 서버 IP주소, 파티션 이름, 해당 문서함의 아이디, 부서 폐쇄 일자가 표시됩니다.
3.  삭제할 부서문서함의 좌측 체크박스를 클릭하고 우측 상단의 삭제 버튼을 클릭합니다.​<br>

{% hint style="warning" icon="square-poll-horizontal" %}
관리자가 별도로 삭제하지 않을 경우, 폐쇄된 부서문서함은 부서 폐쇄일로부터 30일 후 서버에서 자동 삭제됩니다.
{% endhint %}
