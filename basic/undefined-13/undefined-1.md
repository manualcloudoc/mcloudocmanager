# 관리자가 웹에서 폴더 용량 제한 현황 확인하는 방법

관리자는 웹에서 부서문서함 내 용량이 제한된 폴더의 상세 사용 현황을 확인할 수 있습니다.&#x20;

1. **일반 모듈 관리 - Basic - 문서함 관리 - 폴더 용량 관리 - 폴더 용량 현황**을 클릭합니다.

![](<../../.gitbook/assets/img_000 (29).png>)

2. 좌측 상단에서 **부서문서함**을 선택합니다.
3. 폴더 목록에서 조회할 **폴더명**을 클릭합니다.

{% hint style="warning" icon="square-poll-horizontal" %}
용량이 제한된 폴더는 <img src="../../.gitbook/assets/img_002 (24).png" alt="" data-size="line">로 표시되며 선택 시 상세 사용 현황을 확인할 수 있습니다. \
용량이 제한되지 않는 폴더는 <img src="../../.gitbook/assets/img_003 (21).png" alt="" data-size="line">와 같이 비활성화되어 선택할 수 없습니다.
{% endhint %}

{% hint style="success" icon="lightbulb-exclamation-on" %}
<img src="../../.gitbook/assets/img_005 (13).png" alt="" data-size="line"> **부서문서함**을 선택하면 부서문서함 전체에 할당된 용량, 사용 중인 공간, 사용 가능한 공간을 확인할 수 있습니다.
{% endhint %}

4. 우측 **폴더 용량 현황** 영역의 내용을 확인합니다.

* **할당 용량**: 폴더에 할당된 용량으로 총 사용 가능한 용량입니다.
* **사용 중인 공간** 또는 **사용량**: 현재 사용 중인 용량입니다.
* **사용 가능한 공간** 또는 **잔여량**: 할당된 용량에서 사용 중인 용량을 제외한 용량입니다.

{% hint style="success" icon="lightbulb-exclamation-on" %}
원형 차트의 **사용량/잔여량** 영역에 마우스 포인트를 올리면 상세 퍼센트(%) 값을 확인할 수 있습니다.   &#x20;
