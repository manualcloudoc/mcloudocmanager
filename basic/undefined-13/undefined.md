# 파일 확장자 필터링으로 파일 업로드/다운로드 등 차단하는 방법

특정 확장자 파일에 대하여 사용자의 파일 업로드/다운로드, 생성/삭제, 읽기 작업을 차단할 수 있습니다.&#x20;

1\.   **일반 모듈 관리 - Basic - 문서함 관리 - I/O 제어 - 파일 확장자 필터링**을 클릭합니다.

![](<../../.gitbook/assets/img_000 (93).png>)\
&#x20; 2\.   **파일 확장자 필터링** 항목에서 사용자들의 문서함 작업을 차단할 확장자의 체크박스를 선택합니다.\
원하는 파일 확장자가 없을 경우 **기타 확장자** 입력란에 파일 확장자명을 직접 입력합니다.

1. **기타 확장자**: 여러 개를 입력 시 \*\*‘,’\*\*로 구분합니다. 입력 예시)  jpg,gif,hwp

3\.   **확인**을 클릭합니다.
