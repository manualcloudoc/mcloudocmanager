# 폴더관리자 신청 처리 및 이력 보기

관리자는 사용자의 폴더관리자 신청 내용을 확인 후 승인 또는 반려할 수 있고, 신청 처리한 이력을 확인할 수 있습니다. &#x20;

## 폴더관리자 신청처리하기

관리자가 사용자의 폴더관리자 신청을 승인 또는 반려하는 방법은 다음과 같습니다.

1\.   \*\*일반 모듈 관리 - Basic - 문서함 관리 - 폴더 권한 관리

* 신청 처리\*\*를 클릭합니다.

![](<../../.gitbook/assets/img_000 (99).png>)

2\.    사용자들의 신청 내역을 확인하고 사용자 선택 후 **승인** 또는 **반려** 버튼을 클릭합니다.

3\.    **‘승인’** 또는 **‘반려’** 창에서 사유를 입력 후 **확인**을 클릭합니다. 승인/반려하면 처리 결과와 사유가 알림 메시지와 이메일로 사용자에게 전송됩니다.

![](<../../.gitbook/assets/img_001 (92).png>)

![Notes](<../../.gitbook/assets/img_002 (77).png>)자세한 폴더관리자 신청처리 결과 확인 방법은사용자 매뉴얼[**폴더관리자 신청하기**](https://github.com/manualcloudoc/mcloudoc-user-manual/blob/main/zoho-export/markdown/%EC%82%AC%EC%9A%A9%EC%9E%90-%EB%A7%A4%EB%89%B4%EC%96%BC/basic/%ED%8F%B4%EB%8D%94%EA%B4%80%EB%A6%AC%EC%9E%90-%EC%8B%A0%EC%B2%AD%ED%95%98%EA%B8%B0.md)를 참고하세요.&#x20;

## 사용자의 폴더관리자 신청 처리 이력 보기

관리자가 사용자의 폴더관리자 신청을 처리한 이력을 확인하는 방법은 다음과 같습니다.

1\.    **일반 모듈 관리 - Basic - 문서함 관리 - 폴더 권한 관리 - 신청 처리 이력**을 클릭합니다.

2\.    \*\*‘신청 처리 이력’\*\*창에서 검색 항목을 입력하고 **검색**을 클릭합니다.

1. **작업 기간**:  사용자가 폴더관리자 신청한 날짜입니다.
2. **문서함**: 신청 대상 폴더를 선택합니다.
3. **작업 종류**:  전체, 승인, 반려 또는 취소를 선택합니다.
4. **검색 조건**:  사용자 ID, 승인권자 ID, 폴더 경로를 입력합니다.

![](<../../.gitbook/assets/img_003 (63).png>)\
3\.   신청 처리 이력 목록에서 폴더관리자 신청 처리 이력을 조회할 수 있습니다. 상세한 승인 정보는 목록의 **보기**버튼을 클릭하여 조회합니다.

![](<../../.gitbook/assets/img_004 (48).png>)
