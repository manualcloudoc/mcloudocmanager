# 사용자 일괄 등록하기

사용자를 일괄로 등록하는 방법은 다음과 같습니다.

1\.   관리자 웹페이지에서 **일반 관리 – 인사 조직 관리 - 사용자 – 사용자 관리**를 선택한 후, 사용자 관리 페이지에서 **일괄 추가** 버튼을 클릭합니다. ![](<../../.gitbook/assets/img_000 (47).png>)

2\.   ‘사용자 일괄등록’ 창에서 **엑셀 파일 다운로드**를 클릭하여 엑셀 양식을 다운로드합니다.

![](<../../.gitbook/assets/img_001 (46).png>)

3\.   다음의 예와 같이, 추가할 모든 사용자에 대해 <사용자 이름, 사용자 ID, 패스워드, 이메일, 휴대전화번호, 직위, 부서코드, 용량> 형식으로 사용자 정보를 입력한 후, 파일을 저장합니다.&#x20;

![](<../../.gitbook/assets/img_002 (41).png>)

1. **사용자 이름(필수)**: 사용자 이름을 빈 칸 없이 입력합니다.
2. **사용자 ID(필수)**: 사용자의 문서중앙화 ID로, 5자 이상 40자 이하로 지정합니다.
3. **패스워드(필수)**: 8자 이상 20자 이하로 지정합니다.
4. **이메일(필수)**: 이메일 주소는 필수로 입력합니다.
5. **휴대전화번호**: 000-0000-0000 형식으로 입력합니다. 휴대폰 번호는 생략 가능합니다.
6. **직위**: 직위 파일에 포함된 직위명으로 입력합니다. 직위는 생략 가능합니다.
7. **부서코드**: 사용자가 소속된 부서의 부서코드로 입력합니다. 부서코드는 조직도(부서) 등록 시 설정한 코드로, 소속 부서를 지정하지 않을 경우 생략 가능합니다.
8. **용량(필수)**: 사용자의 개인문서함에 할당할 용량을 MB 단위로 입력합니다.

![Notes](<../../.gitbook/assets/img_003 (36).png>)첫 행은 항목명 등 추가 정보를 입력하는 용도로 사용되며 실제 데이터로는 인식되지 않습니다.

4. ‘사용자 일괄등록’ 창에서 **업로드** 버튼을 클릭하여 위에서 작성한 파일을 선택한 후, **등록** 버튼을 클릭합니다.

![](<../../.gitbook/assets/img_004 (29).png>)

5\.   등록에 성공하면 아래와 같이 성공 메시지(등록한 사용자 수)가 표시되고, 사용자가 서버에 추가됩니다. 이때 사용자의 개인문서함이 생성되며, 조직도에 사용자의 소속 정보가 등록됩니다.

![](<../../.gitbook/assets/img_005 (25).png>)

1. 오류가 있는 경우 사유가 아래와 같이 표시됩니다. 이 경우, 오류 데이터를 수정하여 정상 등록된 데이터를 제외하고 재등록합니다.

![](<../../.gitbook/assets/img_006 (21).png>)

6\.   등록된 사용자 정보는 사용자 관리 메뉴나 조직도 편집기에서 확인할 수 있습니다.

1. **인사 조직 관리 – 사용자 – 사용자 관리**에서 등록된 사용자 정보를 확인할 수 있습니다.

![](<../../.gitbook/assets/img_007 (19).png>)

1. 조직도 편집기의 좌측 부서 트리에서 부서를 선택하여 부서에 속한 부서원을 확인할 수 있습니다.

![](<../../.gitbook/assets/img_008 (17).png>)
