# 직위 일괄 등록하기

조직도에 직위를 일괄로 등록하는 방법은 다음과 같습니다.

1. 관리자 웹페이지에서 **일반 관리 – 인사 조직 관리 - 조직도 – 조직도 관리**를 선택한 후, 조직도 관리 페이지에서 **직위 일괄등록** 우측의 **일괄 추가** 버튼을 클릭합니다.&#x20;

<figure><img src="../../.gitbook/assets/img_000 (88).png" alt=""><figcaption></figcaption></figure>

2. ‘직위 일괄등록’ 창에서 **엑셀 파일 다운로드**를 클릭하여 엑셀 양식을 다운로드합니다.&#x20;

<div align="left"><figure><img src="../../.gitbook/assets/img_001 (82).png" alt="" width="354"><figcaption></figcaption></figure></div>

3. 추가할 직위명을 다음의 예와 같이 모두 입력한 후, 파일을 저장합니다.             &#x20;

<div align="left"><figure><img src="../../.gitbook/assets/img_002 (69).png" alt=""><figcaption></figcaption></figure></div>

{% hint style="warning" icon="square-poll-horizontal" %}
첫 행은 항목명 등 추가 정보를 입력하는 용도로 사용되며 실제 데이터로는 인식되지 않습니다.
{% endhint %}

4. ‘직위 일괄등록’ 창에서 **업로드** 버튼을 클릭하여 위에서 작성한 파일을 선택한 후 **등록** 버튼을 클릭합니다.

![](<../../.gitbook/assets/img_004 (43).png>)

5. 등록에 성공하면 아래와 같이 성공 메시지(등록한 직위 개수)가 표시되고, 등록한 직위가 서버의 조직도에 추가됩니다. 전체가 성공하지 못한 경우, 등록되지 못한 데이터만 수정하여 다시 등록합니다.

<div align="left"><img src="../../.gitbook/assets/img_005 (36).png" alt="" width="532"></div>

* 오류가 있는 경우 사유가 표시됩니다. 이 경우, 오류 데이터를 수정하여 정상 등록된 데이터를 제외하고 재등록합니다.

6. 직위 등록 내용은 **조직도 편집기**를 실행하여 확인할 수 있습니다.

* 조직도 편집기에서 **직위 관리** 메뉴를 선택하면 등록된 직위 목록을 확인할 수 있습니다.

![](<../../.gitbook/assets/img_006 (30).png>)
