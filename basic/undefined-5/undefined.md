# 사용자 추가, 삭제, 정보 변경, 검색하기

## 사용자 추가하기

사용자 추가는 다음 세 가지 경로로 가능합니다.

* 관리자가 **사용자 관리** 메뉴에서 사용자를 추가
* 관리자가 **조직도** **편집기**를 실행하여 사용자를 추가
* 사용자가 직접 웹에서 가입 신청(관리자 승인 설정 가능)

{% hint style="warning" icon="square-poll-horizontal" %}
솔루션을 기존 조직도와 연동한 경우 별도의 사용자 추가 작업이 필요하지 않습니다.
{% endhint %}

위의 세 가지 방법 중 여기서는 관리자가 **사용자 관리** 메뉴에서 사용자를 추가하는 경우에 대해 설명합니다.&#x20;

1. **일반 관리** **-** **인사 조직 관리 - 사용자** **-** **사용자 관리**를 클릭합니다.
2. **추가**를 클릭합니다.&#x20;
3. **기본 정보**를 입력합니다.&#x20;

6. 신규 사용자를 추가한 후에 관리자는 해당 로그인 정보를 사용자에게 전달합니다. 사용자는 전달받은 정보(사용자ID, 초기 비밀번호)로 시스템에 로그인합니다.

사용자 목록 상단에서 현재 사용 중인 사용자 수와 시스템에 추가 가능한 사용자 수를 확인할 수 있습니다.
{% endhint %}

### <mark style="color:$primary;">사용자 삭제하기</mark>

사용자를 삭제하면 더 이상 해당 계정으로 솔루션에 접속할 수 없고, 해당 계정의 개인문서함에 있는 데이터도 모두 삭제됩니다.&#x20;

1. **일반관리 - 인사 조직 관리 - 사용자 - 사용자 관리**를 클릭합니다.
2. 삭제할 사용자를 검색합니다.
3. 목록에서 해당 사용자 계정의 **휴지통**을 클릭합니다.&#x20;

{% hint style="danger" %}
사용자 계정 삭제 전에 [**사용자**](undefined-3.md)[**개인문서함의 데이터 인계하기**](undefined-3.md) 기능을 통해 개인문서함 내의 모든 데이터를 다른 사용자에게 인계하면 회사의 자산인 데이터를 보존할 수 있습니다. \
&#x20;
{% endhint %}

### <mark style="color:$primary;">사용자 개인정보 변경하기</mark>

관리자는 사용자 계정의 기본 정보와 비밀번호를 변경할 수 있습니다.&#x20;

#### 기본 정보 변경하기

1. **일반 관리** **-** **인사 조직 관리 - 사용자** **-** **사용자 관리**를 클릭합니다.
2. 목록에서 사용자의 **연필** 아이콘을 클릭합니다.**사용자 계정 관리** 페이지로 이동합니다.
3. **기본 정보** 탭에서 기본정보 및 상태를 변경할 수 있습니다.

* **기본 정보**: 이름, 이메일, 생년월일, 휴대 전화번호 등을 변경할 수 있습니다. ID는 변경할 수 없습니다.
* **계정 상태**: 정상 또는 정지를 선택할 수 있습니다. 정지 상태인 경우에는 로그인할 수 없으므로 계정을 사용하려면 정상 상태로 변경해야 합니다.

4. **확인**을 클릭합니다.

{% hint style="warning" icon="square-poll-horizontal" %}
사용자가 비밀번호 입력 횟수를 초과한 경우 더 이상 로그인할 수 없도록 사용자 계정을 **정지** 상태로 변경할 수 있습니다. 정지 상태의 사용자는 이메일 인증을 통해 비밀번호를 변경하여도 로그인할 수 없습니다. 반드시 관리자가 계정 상태를 **정상**으로 설정해 주어야만 로그인이 가능합니다.&#x20;
{% endhint %}

#### 사용자 비밀번호 변경하기

관리자가 사용자의 비밀번호를 직접 변경하는 방법은 다음과 같습니다.

1. **일반 관리** **-** **인사 조직 관리 - 사용자** **-** **사용자 관리**를 클릭합니다.
2. 목록에서 사용자의 **연필** 아이콘을 클릭합니다. **사용자 계정 관리** 페이지로 이동합니다.
3. **비밀번호** 탭을 클릭합니다.
4. 새 비밀번호를 입력 후 **확인**을 클릭합니다.

### <mark style="color:$primary;">사용자 정보 검색하기</mark>

계정 상태 및 사용자의 이름, ID로 사용자의 계정을 검색할 수 있습니다.

1. **일반 관리**-**인사 조직 관리 - 사용자**-**사용자 관리**를 클릭합니다.
2. **계정 상태**를 선택합니다.
3. 필요에 따라 검색 조건(사용자 이름, 사용자 ID)을 입력 후 **검색**을 클릭합니다.
4. 목록에서 사용자 정보를 확인합니다.

* **마지막 로그인:** 사용자가 문서중앙화 시스템에 마지막으로 로그인한 날짜와 시간을 표시합니다.
* **최종 접속:** 오늘 기준으로 며칠 전에 사용자가 마지막으로 접속하였는지 표시합니다. (하루 미만인 경우 분, 시로 표시)
* **구분:** 사용자 계정은 생성 방법에 따라 아래와 같이 구분합니다.

> - **일반**: 문서중앙화 솔루션에서 생성한 계정
> - **SSO**: 그룹웨어 등 연동한 프로그램에 의해 생성된 계정

* **계정 상태**:

> - **정상**: 사용자는 정상 상태에서만 로그인을 할 수 있습니다.
> - **정지**: 정지 상태에서는 로그인할 수 없습니다.
