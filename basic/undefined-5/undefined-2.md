# 사용자 계정 관리에서 모바일 기기 관리하기

이미 등록된 모바일 기기에 대한 세부 정보를 조회하고, 2단계 인증 승인용 기기를 설정하거나 등록된 모바일 기기를 제거할 수 있습니다. 휴대폰 분실 등의 경우에는 분실신고 상태로 변경하여 서버 접속을 차단할 수 있습니다.휴대폰 분실 등의 경우에는 **분실신고** 상태로 변경하여 서버 접속을 차단할 수 있습니다.

{% hint style="warning" icon="square-poll-horizontal" %}
모바일 기기 등록 방법은 [**모바일 앱 시작하기 및 매뉴얼 가이드**](https://app.gitbook.com/s/vQ0BiQsGY4PT08D7Nfay/mobile)**,** [**모바일 앱에서 디바이스 등록하기**](https://app.gitbook.com/s/vQ0BiQsGY4PT08D7Nfay/mobile/undefined/undefined-3)를 참고하세요.
{% endhint %}

#### <mark style="color:$primary;">사용자의 모바일 기기 세부정보 확인하는 방법</mark>

1. **일반관리**-**인사 조직 관리 - 사용자-사용자 관리**를 클릭합니다.
2. 목록에서 사용자의 **연필**아이콘을 클릭합니다. **사용자 계정 관리**페이지로 이동합니다.
3. 좌측의 **모바일 기기 관리**탭메뉴를 클릭합니다.&#x20;

![](<../../.gitbook/assets/img_001 (45).png>)

4. **세부정보 더보기**를 클릭합니다.

<div align="left"><img src="../../.gitbook/assets/img_002 (40).png" alt="" width="563"></div>

5. **‘세부정보 더보기’** 창에서 등록된 모바일의 기종, 디바이스 ID, 설치된 소프트웨어의 버전, 기기 상태를 확인할 수 있습니다.

#### <mark style="color:$primary;">사용자의 모바일 기기 분실신고하는 방법</mark>

사용자가 모바일 기기를 분실한 경우 분실신고를 통해 해당 기기의 서버 접속을 차단할 수 있습니다.

1. **모바일 기기 관리** 페이지로 이동합니다.
2. 분실한 기기의 **세부정보 더보기**를 클릭합니다.&#x20;
3. &#x20;기기 **상태**를 정상에서 **분실신고**로 변경합니다.

* **상태**: 모바일 기기의 서버 접속을 허용 또는 차단할 수 있습니다.

> - **정상:** 해당 모바일 기기의 서버 접속을 허용합니다.
> - **분실신고:** 해당 모바일 기기에 저장된 파일을 삭제하고 서버 접속을 차단합니다.

#### <mark style="color:$primary;">2단계 인증 승인용 기기를 변경하는 방법</mark>

등록한 모바일 기기가 여러 개일 경우 2단계 인증 승인용 기기를 변경하는 방법은 다음과 같습니다.

1. **모바일 기기 관리** 페이지로 이동합니다.
2. 승인용 기기로 설정할 모바일 기기 항목의 **더보기** <img src="../../.gitbook/assets/img_003 (35).png" alt="" data-size="line">버튼을 클릭합니다.
3. **2단계 인증 승인용 기기로 사용**을 선택합니다.&#x20;

![](<../../.gitbook/assets/img_004 (28).png>)

#### <mark style="color:$primary;">​모바일 기기 등록 해제하는 방법</mark>

1. **모바일 기기 관리** 페이지로 이동합니다.
2. 기기 등록을 해제할 모바일 기기 항목의 **더보기** <img src="../../.gitbook/assets/img_005 (24).png" alt="" data-size="line">버튼을 클릭합니다.
3. **제거**를 선택하면 모바일 기기 등록이 해제됩니다. 사용자는 이 기기로 서버에 더 이상 접속할 수 없습니다.
