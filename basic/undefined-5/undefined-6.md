# 사용자 정보 변경 이력 보기

현재 등록된 사용자 뿐만 아니라 삭제된 사용자까지도 조회 가능합니다.

1. **일반 관리**-**인사 조직 관리**-**사용자-사용자 정보 변경 이력**을 클릭합니다.
2. 검색 항목을 입력합니다.

* **작업 종류 (필수)**: 작업 종류는 다음과 같이 구분합니다.

> - **정보 열람** : 사용자 정보를 조회
> - **정보 변경** : 사용자 정보를 변경
> - **사용자 생성** : 사용자 ID를 추가​
> - **사용자 탈퇴** : 사용자 ID를 삭제

* **검색 조건**

> - **사용자**의**ID** 또는 **이름** 혹은 **작업한 사용자**의 **ID** 또는 **이름을** 입력합니다.

3. **검색**을 클릭합니다.
4. 목록에서 작업 내용과 작업한 계정에 대한 이력을 조회할 수 있습니다.
