# 사용자의 가입 신청 처리하기

### <mark style="color:$primary;">사용자의 가입 신청 처리하기</mark>

이 기능은 고객이 별도로 요구한 경우에만 활성화합니다. 웹에서 직접 ‘**사용자 가입 신청**’을 한 경우 관리자가 승인할 수 있습니다. 설정에 따라 관리자의 승인 없이 사용자 가입 신청만으로 등록도 가능합니다. 가입 신청 시 필요한 입력 항목은 관리자가 **시스템 설정** **-** **설정 홈 연결** **-** **일반 설정** **-** **인사 조직 관리**- **사용자** **- 회원 정보 입력 항목 설정**에서 설정할 수 있습니다.

#### 사용자의 가입 신청에 대하여 승인/반려하는 방법

1. **일반관리** **-** **인사 조직 관리 -  사용자 -  가입 신청 처리**를 클릭합니다.

![](<../../.gitbook/assets/img_000 (52).png>)

2. **가입 신청 처리** 페이지에서 신청 목록을 조회합니다.
3. 목록에서 승인/반려할 사용자의 좌측 체크박스를 선택합니다.
4. **승인/반려**를 클릭하여 승인/반려합니다.
5. 승인/반려하면 신청한 사람에게 이메일로 처리 결과가 전송됩니다.

#### 가입 신청한 사용자의 정보 상세보기

1. **일반관리 - 인사 조직 관리 -  사용자 -  가입 신청 처리**를 클릭합니다.
2. **가입 신청 처리** 페이지에서 신청 목록을 조회합니다.
3. 목록에서 승인/반려할 사용자의 **보기**버튼을 선택합니다.
4. **'상세보기’** 창에서 사용자가 가입 신청할 때 작성한 정보를 확인할 수 있습니다.&#x20;

### <mark style="color:$primary;">사용자 가입 신청 처리 이력보기</mark>

사용자의 가입 신청 이력 및 그에 대한 처리 이력을 확인할 수 있습니다.

1. **일반관리** **-** **인사 조직 관리 -  사용자 -  가입 신청 처리 이력**을 클릭합니다.
2. **신청 기간, 작업 종류**를 선택하고 필요에 따라 검색 조건을 입력합니다.
3. **검색**을 클릭합니다.
