# 웹에서 사용자의 개인문서함 용량 변경하기

사용자 개인문서함의 용량을 관리자가 추가, 삭제, 변경할 수 있습니다.&#x20;

### <mark style="color:$primary;">용량 추가하기</mark>

1. **일반관리 - 인사 조직 관리 - 사용자 - 사용자 관리**를 클릭합니다.
2. 목록에서 사용자의 **연필** 아이콘을 클릭합니다. **사용자 계정관리** 페이지로 이동합니다.
3. **용량 상세 목록**으로 이동합니다.
4. **추가**를 클릭합니다.

<figure><img src="../../.gitbook/assets/img_000 (65).png" alt=""><figcaption></figcaption></figure>

5. **‘추가’** 창에 정보를 입력합니다.

* **용량**: 추가할 용량을 입력합니다. 용량 단위는 MB입니다
* **사용 만료일**: 추가 용량의 사용 기한을 입력합니다.

6. **확인**을 클릭합니다.
7.

다음 동영상에서 개인문서함 용량을 추가하고, 변경된 용량을 확인인하는 방법을 참고합니다.

### <mark style="color:$primary;">용량 변경하기</mark>

개인문서함의 용량, 사용 만료일 등을 변경할 수 있고, 용량 상세 목록에서 이력을 조회할 수 있습니다.&#x20;

1. **일반관리**-**인사 조직 관리**-**사용자**-**사용자 관리**를 클릭합니다.
2. 목록에서 사용량을 변경할 사용자의 **연필** 아이콘을 클릭합니다. **사용자 계정관리** 페이지로 이동합니다.
3. **용량 상세 목록**으로 이동합니다.
4. 용량 상세 목록에서 변경할 용량의 **연필** 아이콘을 클릭합니다.
5. **‘수정’** 창에 변경할 정보를 입력합니다.

<div align="left"><img src="../../.gitbook/assets/img_002 (51).png" alt=""></div>

6. **확인**을 클릭합니다.

### <mark style="color:$primary;">용량 삭제하기</mark>

용량 추가 이력을 목록에서 조회할 수 있습니다. 목록에서 해당 추가 건을 삭제하면 그 용량을 더 이상 사용할 수 없습니다.

1. **일반관리**-**인사 조직 관리**-**사용자**-**사용자 관리**를 클릭합니다.
2. 목록에서 사용자의 **연필** 아이콘을 클릭합니다. **사용자 계정관리** 페이지로 이동합니다.
3. **용량 상세 목록**으로 이동합니다.
4. 용량 상세 목록에서 삭제할 용량의 **휴지통**을 클릭합니다.
5. 팝업 메시지 창에서 **확인**을 클릭합니다.
