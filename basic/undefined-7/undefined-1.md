# 조직도에서 사용자 추가 및 관리하기

{% hint style="info" %}
메뉴 경로: **일반 관리 - 인사 조직 관리 - 조직도 - 조직도 관리 - 조직도 편집기 실행** 버튼 클릭
{% endhint %}

조직도 편집기 내 특정 위치(부서)에 새로운 사용자를 생성하여 추가하거나 기존의 사용자를 추가할 수 있습니다. 또한, 사용자의 정보 변경, 삭제 및 탈퇴가 가능합니다.

#### 새 사용자 추가하기

새로운 사용자를 생성하여 조직도에 추가합니다. 이때 해당 사용자의**개인문서함**도 더불어 생성됩니다.\
&#x20;

1. 조직도의 **부서/그룹** 트리에서 새 사용자가 소속될 부서를 선택합니다.

{% hint style="warning" icon="square-poll-horizontal" %}
부서/그룹 트리에서 **전체 사용자**를 선택한 후 생성한 사용자는 어떠한 부서에도 소속되지 않습니다.
{% endhint %}

2. 주 메뉴의 **조직도 - 새 사용자**를 선택합니다.
3. ‘**사용자 추가**’ 창에서 **이름, 아이디, 암호, 암호 확인, 전자 메일 주소**를 필수적으로 입력합니다.
4. 개인문서함 정보에서 **서버 위치, 중요도, 용량, 연결할 드라이브**를 필수적으로 선택, 입력합니다.

{% hint style="warning" icon="square-poll-horizontal" %}
**아이디, 암호, 전자 메일 주소** 및 **개인문서함 정보** 입력 값에 대한 설명은 [**사용자 추가, 삭제, 정보 변경, 검색하기**](../undefined-5/undefined.md)를 참고하세요.
{% endhint %}

5. 기타 사용자 정보는 선택적으로 입력합니다.
6. **추가**를 클릭하면 해당 위치에 사용자가 추가됩니다. 창이 열린 상태에서 같은 방법으로 계속해서 새로운 사용자를 추가할 수 있습니다.
7. **닫기**를 클릭하면 생성된 사용자 목록이 반영되어 화면에 표시됩니다.

![](<../../.gitbook/assets/img_003 (52).png>)

#### <mark style="color:$primary;">사용자의 직위 설정하기</mark>

직위 관리에서 등록된 직위를 사용하여 사용자의 직위를 설정할 수 있습니다.

1. 직위를 부여할 사용자를 선택합니다.
2. 주 메뉴의 **편집 - 속성**을 선택합니다.
3. ‘<사용자> 속성’ 창에서 **직위** 항목의 드롭다운 리스트를 클릭하여 사용자에게 부여할 직위를 선택합니다.
4. **확인**을 클릭하여 변경 사항을 적용합니다.  <br>

<figure><img src="../../.gitbook/assets/img_004 (41).png" alt=""><figcaption></figcaption></figure>

#### <mark style="color:$primary;">기존 사용자 추가하기</mark>

이미 생성한 사용자를 특정 부서에 소속시킬 수 있습니다.&#x20;

1. 조직도의 **부서**/**그룹** 트리에서 사용자가 소속될 부서를 선택합니다.
2. 주 메뉴의 **조직도 - 기존 사용자 추가**를 선택합니다.
3. ‘**검색 후 사용자 선택**’ 창에서 **검색할 이름**을 입력한 후 **검색**을 클릭합니다.
4. **검색 결과**에서 추가할 사용자를 선택한 후 **추가**를 클릭합니다. 같은 방법으로 계속해서 사용자를 검색하여 동일한 위치에 추가할 수 있습니다.
5. **닫기**를 클릭하면 새롭게 소속된 사용자 목록이 반영되어 화면에 표시됩니다.

![](<../../.gitbook/assets/img_005 (35).png>)

#### <mark style="color:$primary;">사용자 정보 변경하기</mark>

사용자의 계정 정보, 개인 정보, 회사 정보, 개인문서함 정보를 변경할 수 있습니다.

1. 정보를 변경할 사용자를 선택합니다.
2. 주 메뉴의 **편집 - 속성**을 선택합니다.
3. ‘**<사용자> 속성**’ 창에서 변경할 내용을 입력합니다. 이때 사용자의 **이름, 아이디, 서버 위치, 중요도 정보**는 변경할 수 없습니다.
4. **확인**을 클릭하여 변경 사항을 적용합니다.![Idea](<../../.gitbook/assets/img_006 (29).png>)사용자 항목을 마우스 더블 클릭해도 **‘**<**사용자**>**속성**’ 창이 열립니다.

![](<../../.gitbook/assets/img_007 (25).png>)

#### <mark style="color:$primary;">사용자 삭제하기</mark>

삭제된 사용자는 해당 부서에서만 제거되며 **전체** **사용자**목록에 남아있습니다. &#x20;

1. 조직도의 **부서**/**그룹** 트리에서 삭제할 사용자의 소속 부서를 선택합니다.
2. 삭제할 사용자를 선택합니다.
3. 주 메뉴의 **편집 - 삭제**를 선택하여 사용자를 부서에서 제거합니다.

{% hint style="warning" icon="square-poll-horizontal" %}
다른 부서에 중복 소속된 사용자를 삭제하는 경우, 다른 부서 소속에는 영향을 주지 않습니다.
{% endhint %}

#### <mark style="color:$primary;">사용자 탈퇴 처리하기</mark>

탈퇴한 사용자는 사용자 목록에서 삭제되며 더 이상 서비스에 로그인할 수 없습니다.  사용자 탈퇴 시 해당 사용자의 개인문서함 내 모든 자료도 삭제됩니다.

1. 탈퇴할**사용자**를 선택합니다.
2. 주 메뉴의 **조직도 - 사용자 탈퇴**를 선택합니다.

{% hint style="danger" %}
탈퇴된 사용자는 다시 복구할 수 없으므로 주의하시기 바랍니다.
{% endhint %}

#### <mark style="color:$primary;">사용자 검색하기</mark>

조직도에서 사용자 이름으로 사용자를 검색하고, 소속된 부서 위치를 **부서/그룹** 트리에서 확인할 수 있습니다.

1. 조직도의 **부서**/**그룹** 트리에서 검색 범위로 사용될 부서를 선택합니다. 이때 **전사 아이콘**을 선택하면 조직 전체에서 사용자를 검색할 수 있습니다.
2. 주 메뉴의 **편집 - 검색**을 선택합니다.
3. ‘<부서명> 에서 검색’ 창에서 **검색할 이름**을 입력한 후 **검색**을 클릭합니다. 이때 이름의 첫 글자부터 일부만 입력해도 검색이 가능합니다.
4. **검색 결과**에 표시된 사용자 목록에서 사용자를 선택하고 **위치로 이동**을 클릭하면 조직도에서 소속 부서의 위치를 확인할 수 있습니다.&#x20;
