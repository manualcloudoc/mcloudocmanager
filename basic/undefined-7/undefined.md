# 조직도 편집기 소개

조직도 편집기를 사용하여 사내 조직도를 계층적으로 만들 수 있습니다. 생성된 조직도는 솔루션 기능 이용 시 다음의 예와 같이 활용됩니다.

1. 조직도 내 부서 단위로 **부서문서함** 생성
2. 부서문서함 폴더의 사용 권한을 조직도 상의 부서, 사용자, 직위 별로 설정
3. 문서반출 등 각종 승인이 필요한 요청의 결재 라인 결정
4. IP 대역 및 각종 보안 관련 정책의 적용 단위

조직도 편집기는 다음과 같은 기능을 제공합니다.

1. **부서 관리**: 새 부서를 생성하고 부서간 상하 관계를 구성할 수 있습니다.
2. **직위 관리**: ‘부장-차장-과장-대리-주임-사원’과 같은 직위체계를 설정합니다.
3. **사용자(직원) 관리**: 사용자(직원)를 생성하고 각 사용자별로 부서를 지정합니다.
4. **승인권자(결재권자) 관리**: 부서마다 승인권자(결재권자)를 지정합니다.
5. **부서문서함 관리**: 전사 또는 부서별 부서문서함을 생성하고, 용량 등을 설정할 수 있습니다.

## 조직도 편집기 실행하기

조직도 편집기를 실행하는 방법은 다음과 같습니다.\
&#x20;   1\.     웹페이지에서 **일반 관리 - 인사 조직 관리 - 조직도 - 조직도 관리**를 클릭합니다.

2\.     **조직도 관리** 페이지에서 **조직도 편집기 실행**버튼을 클릭합니다.

## 조직도 편집기 화면 구성

조직도 편집기 화면은 아래와 같이 구성됩니다.

![](<../../.gitbook/assets/img_001 (74).png>)

①  **상단 메뉴**: 조직도 편집을 위한 **메뉴**와 **빠른 실행 도구 모음**을 표시합니다.\
②  **좌측 부서/그룹 트리**: 부서를 트리 구조로 시각화하여 표시합니다.  전체 사용자를 선택하면 모든 사용자 목록을 표시합니다.

③  **우측 리스트**: 현재 트리에서 선택한 부서의 하위 부서와 소속된 사용자 목록을 표시합니다.

④  **하단** **상태표시줄**: 선택한 메뉴 기능의 도움말을 제공하고 키보드 상태를 표시합니다.

## 주 메뉴

조직도 편집기의 주 메뉴 항목 및 기능은 다음과 같습니다.

|                   |                                                   |                                  |
| ----------------- | ------------------------------------------------- | -------------------------------- |
| **메뉴**            | **메뉴 항목**                                         | **기능**                           |
| **조직도**           | **새 사용자**                                         | 새 사용자를 생성한 후 현재 선택한 부서에 소속시킵니다.  |
| **새 부서**          | 현재 선택한 부서의 하위 부서를 새롭게 생성합니다.                      |                                  |
| **기존 사용자 추가**     | 현재 선택한 부서에 기존 사용자를 검색하여 소속시킵니다.                   |                                  |
| **사용자 탈퇴**        | 현재 선택한 사용자가 더 이상 서비스에 로그인 할 수 없도록 탈퇴시킵니다.         |                                  |
| **닫기**            | 조직도 편집기를 종료합니다.                                   |                                  |
| **편집**            | **잘라내기**                                          | 선택한 부서 및 사용자를 잘라낸 후 클립보드에 저장합니다. |
| **복사**            | 선택한 사용자를 복사한 후 클립보드에 저장합니다.                       |                                  |
| **붙여넣기**          | 클립보드에 저장된 부서 또는 사용자를 현재 선택된 부서에 소속시킵니다.           |                                  |
| **삭제**            | 선택한 부서와 그 하위 항목들을 모두 삭제합니다.  사용자는 소속된 부서에서 제거됩니다. |                                  |
| **이름 바꾸기**        | 선택한 부서 이름을 바꿉니다.                                  |                                  |
| **검색**            | 선택한 부서와 그 하위 부서에 소속된 사용자를 검색합니다.                  |                                  |
| **속성**            | 선택한 부서 또는 사용자의 정보를 표시합니다.                         |                                  |
| **보기**            | **도구 모음**                                         | 빠른 실행 도구 모음을 보이거나 숨깁니다.          |
| **상태 표시줄**        | 상태 표시줄을 보이거나 숨깁니다.                                |                                  |
| **큰 아이콘**         | 화면 우측 리스트 항목을 큰 아이콘으로 표시합니다.                      |                                  |
| **작은 아이콘**        | 화면 우측 리스트 항목을 작은 아이콘으로 표시합니다.                     |                                  |
| **간단히**           | 화면 우측 리스트 항목을 간단히 표시합니다.                          |                                  |
| **자세히**           | 화면 우측 리스트 항목을 자세히 표시합니다.                          |                                  |
| **새로 고침**         | 조직도 내용을 최신 상태로 다시 표시합니다.                          |                                  |
| **도구**            | **직위 관리**                                         | 직위 정보를 관리(추가, 삭제, 이름 변경 등)합니다.   |
| **팀장 및 문서관리자 지정** | 선택한 부서 내 문서 보안을 담당할 팀장 및 팀문서관리자를 지정합니다.           |                                  |
| **부서문서함 생성**      | 선택한 부서의 문서함을 생성합니다.                               |                                  |
| **부서문서함 관리**      | 선택한 부서의 문서함 정보 변경하거나 문서함 자체를 삭제합니다.               |                                  |
| **도움말**           | **프로그램 정보**                                       | 조직도 편집기 프로그램 정보를 표시합니다.          |

## 컨텍스트 메뉴

조직도를 구성하는 각 부서 또는 사용자 항목을 마우스 우클릭 하면 컨텍스트 메뉴를 사용할 수 있습니다. 컨텍스트 메뉴를 사용하면, 선택한 부서/사용자에 적용할 주 메뉴의 주요 기능을 곧바로 편리하게 실행할 수 있습니다.
