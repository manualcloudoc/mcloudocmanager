# 조직도 편집기 소개

조직도 편집기를 사용하여 사내 조직도를 계층적으로 만들 수 있습니다. 생성된 조직도는 솔루션 기능 이용 시 다음의 예와 같이 활용됩니다.

* 조직도 내 부서 단위로 **부서문서함** 생성
* 부서문서함 폴더의 사용 권한을 조직도 상의 부서, 사용자, 직위 별로 설정
* 문서반출 등 각종 승인이 필요한 요청의 결재 라인 결정
* IP 대역 및 각종 보안 관련 정책의 적용 단위

조직도 편집기는 다음과 같은 기능을 제공합니다.

* **부서 관리**: 새 부서를 생성하고 부서간 상하 관계를 구성할 수 있습니다.
* **직위 관리**: ‘부장-차장-과장-대리-주임-사원’과 같은 직위체계를 설정합니다.
* **사용자(직원) 관리**: 사용자(직원)를 생성하고 각 사용자별로 부서를 지정합니다.
* **승인권자(결재권자) 관리**: 부서마다 승인권자(결재권자)를 지정합니다.
* **부서문서함 관리**: 전사 또는 부서별 부서문서함을 생성하고, 용량 등을 설정할 수 있습니다.

#### <mark style="color:$primary;">조직도 편집기 실행하기</mark>

조직도 편집기를 실행하는 방법은 다음과 같습니다.

1. 웹페이지에서 **일반 관리 - 인사 조직 관리 - 조직도 - 조직도 관리**를 클릭합니다.
2. **조직도 관리** 페이지에서 **조직도 편집기 실행**버튼을 클릭합니다.

#### <mark style="color:$primary;">컨텍스트 메뉴</mark>

조직도를 구성하는 각 부서 또는 사용자 항목을 마우스 우클릭 하면 컨텍스트 메뉴를 사용할 수 있습니다. 컨텍스트 메뉴를 사용하면, 선택한 부서/사용자에 적용할 주 메뉴의 주요 기능을 곧바로 편리하게 실행할 수 있습니다.
