# 조회한 자료를 엑셀 파일로 저장하는 방법

관리자 웹페이지에서 각종 작업 이력, 현황, 로그 등의 조회 시 화면에 표시된 검색 결과를 다음과 같은 방법으로 엑셀 파일에 저장할 수 있습니다. 여기서는 **2단계 인증 요청 처리 이력**페이지(관리자 웹페이지 메뉴 중 **일반 모듈 관리 - Basic - 로그인 공통 제어 - 2단계 인증 - 요청 처리 이력**)의 예를 들어 설명합니다.

1. 페이지 상단에 조회할 조건을 선택하여 입력한 후 **검색** 버튼을 클릭하면, 하단에 검색 결과가 표시됩니다.

![](<../../.gitbook/assets/img_000 (84).png>)

2. 검색 결과 우측 상단의 **엑셀** 아이콘 ![](<../../.gitbook/assets/img_001 (78).png>)을 클릭한 후, ‘**엑셀파일로 저장하기**’ 창에서 저장할 범위를 선택하고 **다운로드** 버튼을 클릭하면, 선택한 범위의 자료가 저장된 엑셀 파일이 다운로드됩니다.&#x20;

![](<../../.gitbook/assets/img_002 (65).png>)
