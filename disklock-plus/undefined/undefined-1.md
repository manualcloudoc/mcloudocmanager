# 보안 해제 신청 처리 및 이력 보기

#### <mark style="color:$primary;">보안 해제 신청 처리하기</mark>

사용자가 보안 해제를 신청한 경우 승인권자 또는 관리자의 승인이 필요합니다. 관리자는 다음과 같이 사용자의 보안 해제 신청을 승인 또는 반려할 수 있습니다.&#x20;

1. **PC 보안 모듈 관리 - 보안 해제 - 신청 처리**를 클릭합니다.

![](<../../.gitbook/assets/img_000 (104).png>)

2. 사용자의 보안 해제 신청 목록을 확인합니다. 승인을 기다리는 보안 해제 신청 건들의 신청일시, 제목, 해제할 보안 정책, 신청자 이름(ID), 부서명, 신청한 보안해제 시작일과 종료일을 확인할 수 있습니다.
3. 승인/반려하는 방법에는 2가지가 있습니다.

* **목록에서 선택하여 승인 또는 반려하는 방법**

> - 승인 또는 반려 처리할 요청 건 왼쪽의 체크 박스에 체크한 후 우측 상단에서 승인 또는 반려 버튼을 클릭합니다. 여러 요청 건, 혹은 신청 목록의 모든 요청 건을 동시에 선택하여 한 번에 승인 또는 반려 처리할 수도 있습니다.

<figure><img src="../../.gitbook/assets/img_001 (97).png" alt=""><figcaption></figcaption></figure>

> * **‘승인’/’반려’** 확인 창에서 **확인**을 클릭합니다.

* **요청 건 별로 상세 내역을 확인한 후 승인 또는 반려하는 방법**

> - 목록에서 승인/반려하고자 하는 요청 건의 **제목** 항목을 클릭하여 상세보기 페이지로 이동합니다.

<figure><img src="../../.gitbook/assets/img_002 (82).png" alt=""><figcaption></figcaption></figure>

> * **상세보기** 페이지에서 해당 보안 해제 신청 건에 대한 자세한 정보를 확인합니다. 페이지 하단에서 **승인** 또는 **반려** 버튼을 클릭합니다.

<figure><img src="../../.gitbook/assets/img_003 (68).png" alt=""><figcaption></figcaption></figure>

> * **‘승인’/’반려’** 창에서 사유를 입력 후 **확인**을 클릭합니다.

<div align="left"><figure><img src="../../.gitbook/assets/img_004 (51).png" alt=""><figcaption></figcaption></figure></div>

4\.    승인/반려가 완료되면 신청자에게 승인/반려 알림 메일이 발송됩니다.

{% hint style="warning" icon="square-poll-horizontal" %}
관리자가 승인/반려 처리하면 사용자는 다음과 같은 메일을 받습니다. (예시)



<img src="../../.gitbook/assets/img_006 (35).png" alt="" data-size="original">
{% endhint %}

{% hint style="warning" icon="square-poll-horizontal" %}
사용자가 보안해제 신청 시 승인권자를 **팀장/팀문서관리자** 중에서 선택할 수 있으며, 자가승인이 허용된 경우 승인권자가 사용자 본인으로 고정됩니다.

문서반출, 링크 공유, 외부DRM 반출, 보안 해제를 위한 자가승인 허용 여부는 관리자 웹페이지의 **일반 관리 -인사 조직 관리 - 승인권자 관리 - 자가 승인 - 정책 관리/정책 적용**에서 사용자 개인별로 설정할 수 있습니다. 자세한 사항은 [**자가승인 정책 관리 및 적용하기**](../../basic/undefined-8/undefined-1.md)를 참고하세요.&#x20;
{% endhint %}

{% hint style="warning" icon="square-poll-horizontal" %}
승인 절차는 고객사의 전자결재 시스템과 연동하여 동작하도록 구성될 수도 있습니다. 관련 사항은 문서중앙화 시스템 담당자와 협의하시기 바랍니다.
{% endhint %}

#### <mark style="color:$primary;">보안 해제 신청 처리 이력 보기</mark>

관리자는 사용자들의 보안 해제 신청 이력 정보를 다음과 같이 조회할 수 있습니다.

1. **PC 보안 모듈 관리 - 보안 해제 - 신청 처리 이력을** 클릭합니다.
2. 검색 항목을 입력합니다.

* **기간 선택**: 사용자가 보안 해제를 신청한 날짜를 기준으로 검색 기간을 설정합니다.
* **결재 방식**: 선결재, 후결재 혹은 전체를 선택합니다.
* **결재 상태**:

> - **대기**: 승인권자의 승인을 기다리는 상태입니다.
> - **승인**: 승인권자가 보안해제를 승인한 상태입니다.
> - **반려**: 승인권자가 보안해제를 반려한 상태입니다.

* **검색 조건**: 부서명, 신청자의 이름 또는 ID, 승인권자의 이름 또는 ID를 입력합니다.
* **자가 결재**를 체크할 경우, 보안 해제 신청자 본인이 승인권자로 지정된 보안 해제 신청 이력이 검색됩니다.

<figure><img src="../../.gitbook/assets/img_009 (20).png" alt=""><figcaption></figcaption></figure>

3. **검색**을 클릭하면 조건에 해당하는 목록이 검색됩니다.

<figure><img src="../../.gitbook/assets/img_010 (16).png" alt=""><figcaption></figcaption></figure>

4. 제목을 클릭하면 보안 해제 신청의 상세 현황을 볼 수 있습니다.

<figure><img src="../../.gitbook/assets/img_011 (11).png" alt=""><figcaption></figcaption></figure>
