# 인쇄 로그 보기

모듈 정책에서 인쇄 로그 정책을 ‘사용’으로 설정하여 적용한 경우, 해당 사용자의 모든 인쇄 작업 이력이 기록됩니다. 인쇄 작업 이력을 확인하는 방법은 다음과 같습니다.​\
1\.    PC**보안 모듈 관리 - DiskLock Plus - 인쇄 로그 - 인쇄 작업 이력**을 클릭합니다.

![](<../../.gitbook/assets/img_000 (107).png>)

2\.    **인쇄 기간**(필수)을 선택하고, 필요 시 **검색 조건**을 입력합니다.

1. 검색 조건: 사용자 ID, 사용자 이름, 부서명, 파일명, 프린터명, 프린터 드라이버, 프린터 IP, 출력 프로세스

3\.    **검색**버튼을 클릭하면 하단에 검색 조건에 해당하는 인쇄 작업의 인쇄일시, 사용자 이름(ID), 부서명, 파일명, 인쇄 페이지 수, 프린터명, 프린터 드라이버, 프린터 IP, 출력 프로세스, 워터마크 출력 여부가 표시됩니다.

4\.    검색 결과 목록에서파일명을 클릭하면 인쇄에 사용된 컴퓨터명, MAC주소, 컴퓨터 계정 등이 포함된 인쇄 상세 내역을 확인할 수 있습니다.

![Notes](<../../.gitbook/assets/img_001 (100).png>)**파일명**은 인쇄하는 애플리케이션이 부여한 인쇄 작업 이름으로 실제 파일명과 다를 수 있습니다. \
![Notes](<../../.gitbook/assets/img_002 (85).png>)**프린터명**은 \*\*인쇄 제어 -\*\***프린터 등록/제거** 메뉴에서 등록된 프린터명으로 표기됩니다. 등록된 정보가 없는 경우에는 프린터 드라이버명으로 기록됩니다.
