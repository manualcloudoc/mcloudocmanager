# 개인정보보호 설정

![Info](<../../.gitbook/assets/img_000 (189).png>)메뉴 경로: 시스템 설정 - 설정 홈 연결 - 일반 모듈 설정 - 개인정보보호

|                                           |                                                                                                                                                                                                                                                                                                                          |           |
| ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------- |
| **개인정보보호 옵션**                             | **내용**                                                                                                                                                                                                                                                                                                                   | **기본값**   |
| **개인정보 검출** **사용 여부**                     | 문서함의 파일에서 개인정보를 검출하도록 설정합니다.                                                                                                                                                                                                                                                                                             | 사용함       |
| **문서 반출 시** **개인정보 검출 여부**                | 윈도우 에이전트에서 문서 반출 신청 시 개인정보를 검출하도록 설정합니다. 문서에서 개인정보가 검출된 경우 **‘반출 신청’** 창의 **반출 목록**에 관련 정보가 표시됩니다.                                                                                                                                                                                                                       | 사용함       |
| **문서 반출 시** **개인정보 검출 용량 제한**             | 대용량 파일인 경우 개인정보 검출 시 시간이 지연됩니다. 이를 방지하기 위해 특정 용량 이상의 파일은 개인정보를 검출하지 않도록 설정할 수 있습니다.**제한 없음**: 문서 반출 시 용량 제한 없이 검출합니다.**제한**: 입력한 용량(단위: MB) 값 이상의 문서는 검출하지 않고 Pass하도록 설정합니다.                                                                                                                                             | 제한 없음     |
| **윈도우 파일 탐색기 내 파일\*\*\*\*개인정보 아이콘 표시 여부** | 개인정보가 검출된 파일에 개인정보 아이콘을 표시하고 파일 속성에 검출 정보를 추가합니다. 사용 설정 시 탐색기 목록 표시 속도가 저하될 수 있습니다.                                                                                                                                                                                                                                      | 사용 안 함    |
| **사용자에게** **리포트 메일 발송 여부**                | 배치(batch) 작업에 의해 개인정보가 검출된 경우, 해당 파일을 생성한 사용자에게 이메일로 검출 현황 리포트를 발송합니다. 이 리포트에는 문서함별로 신규 검출된 파일/항목의 개수, 검출 후 미처리 파일/항목의 누적 건수가 포함됩니다.                                                                                                                                                                                     | 사용함       |
| **팀장에게** **리포트 메일 발송 여부**                 | 배치(batch) 작업에 의해 개인정보가 검출된 경우, 해당 파일을 생성한 사용자의 팀장에게 이메일로 검출 현황 리포트를 발송합니다. 이 리포트에는 사용자별, 문서함별로 신규 검출된 파일/항목의 개수, 검출 후 미처리 파일/항목의 누적 건수가 포함되어 팀장은 소속 팀원들의 개인정보 검출 현황을 확인하고 대응할 수 있습니다.                                                                                                                                    | 사용함       |
| **통합 리포트 메일** **수신자 설정**                  | 개인정보 검출 내역 **통합 리포트 메일**을 수신할 이메일 계정을 설정합니다. 관리자 메일 계정을 선택하거나 수신할 메일 주소를 직접 입력하여 추가할 수 있습니다. 1. **사용 안 함**: 통합 리포트 메일을 발송하지 않습니다. 2. **관리자 메일 계정**: 메일 서버 관리자의 이메일 계정으로 통합 리포트 메일이 발송됩니다. 메일 서버 관리자의 메일 주소는 **시스템 설정 - 설정 홈 연결 - 고급 설정 - 메일 서비스**에서 설정할 수 있습니다. 3. **통합 리포트 메일 수신자 주소**: 수신할 이메일 주소를 직접 입력하여 추가할 수 있습니다. | 관리자 메일 계정 |
| **메일 언어** **설정**                          | 발송하는 메일의 언어를 설정할 수 있습니다. 1. **한국어** 2. **영어** 3. **일본어** 4. **중국어**                                                                                                                                                                                                                                                      | 한국어       |
