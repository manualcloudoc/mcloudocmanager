# 문서 보존 기한 설정

![Info](<../../.gitbook/assets/img_000 (201).png>)메뉴 경로: 시스템 설정 - 설정 홈 연결 - 일반 모듈 설정 – 문서 보존 기한

|                                    |                                                                                                                       |              |
| ---------------------------------- | --------------------------------------------------------------------------------------------------------------------- | ------------ |
| **문서보안등급 옵션**                      | **내용**                                                                                                                | **기본값**      |
| **문서 보존 기한 사용 여부**                 | 문서 보존 기한 사용 여부를 설정합니다. 사용함으로 설정하면 파일별로 문서 보존 기한 설정이 가능합니다. 폴더에 문서 보존 기한을 지정하면 폴더 내에 저장되어 있는 파일은 자동으로 문서 보존 기한이 설정됩니다. | 사용함          |
| **보존 기한 설정시 사용할 언어 선택**            | 문서 보존 기한 사용 시, 사용할 언어를 선택합니다. 한국어/영어/일본어/중국어                                                                          | 한국어          |
| **보존 기한 선택 옵션**                    | 선택 옵션으로 사용할 보존 기한을 추가/삭제하고 보존 기한 별 사유(용도에 대한 설명)를 설정합니다.                                                              | 1개월, 6개월, 1년 |
| **보존 기한 선택 옵션 추가 여부 - 사용자 직접 입력**  | 보존 기한을 사용자가 직접 입력할 수 있게 하는 기능 추가 여부                                                                                   | 추가함          |
| **문서 보존 기한 변경 신청 메뉴 일반 사용자 사용 여부** | 일반 사용자가 일반 문서에 보존 기한을 설정하거나 기존 보존 기한을 변경할 수 있도록 신청 메뉴가 표시됩니다.                                                         | 사용함          |
| **문서 보존 기한 변경 신청 사용자 파일 권한**       | 문서 보존 기한을 변경 신청하기 위한 사용자의 파일별 최소 권한입니다. 사용자가 최소 권한을 충족한 경우에만 문서 보존 기한 변경 신청이 가능합니다.                                   | 읽기, 삭제, 편집   |
| **문서 보존 기한이 지난 파일의 열람 가능 여부**      | 문서 보존 기한이 지난 파일을 삭제하지 않을 경우, 보존 기한이 지난 해당 파일들에 대한 열람 여부를 결정합니다.                                                       | 열람 가능함       |
