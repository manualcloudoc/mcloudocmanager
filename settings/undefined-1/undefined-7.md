|                                |                                                                                                                   |         |
| ------------------------------ | ----------------------------------------------------------------------------------------------------------------- | ------- |
| **문서함 마운트 해제 옵션**              | **내용**                                                                                                            | **기본값** |
| **마운트 해제 드라이브 타입**             | 서버로 통신이 되지 않거나 해제 설정 시, 드라이브의 마운트가 해제되도록 설정할 수 있습니다. 다음 문서함 별로 설정합니다. 1. **개인문서함** 2. 부서문서함 3. 공유문서함 4. **보안문서함** | 사용 안 함  |
| **마운트 해제 시 등록정보 제거하는 드라이브 타입** | 마운트 해제 시 문서함 내 사용이력 레지스트리 및 탐색기의 즐겨찾기를 삭제하도록 설정할 수 있습니다. 다음 문서함 별로 설정합니다. 1. 개인문서함 2. 부서문서함 3. 공유문서함 4. **보안문서함** | 사용 안 함  |
| **마운트 해제 여부 체크 주기**            | 드라이브 마운트 해제 여부를 체크하는 주기를 설정할 수 있습니다. 1. \*\*사용:\*\*주기(단위:일)를 입력합니다. 2. 사용 안 함                                     | 사용 안 함  |
| **마운트 해제 여부 체크 실패 횟수**         | 설정한 횟수 이상 서버와 통신 실패 시 마운트가 해제되도록 설정할 수 있습니다. 1. **마운트 해제 여부 체크 실패 횟수**: 횟수를 입력합니다. 2. **사용 안 함**                  | ​ 3(회)  |
| **마운트 강제 해제**                  | 설정된 마운트 해제 체크 주기 이후에 로그인한 윈도우 에이전트의 문서함 마운트를 강제로 해제 또는 유지하도록 설정합니다. 1. 마운트 유지 2. 마운트 해제                           | 마운트 유지  |
