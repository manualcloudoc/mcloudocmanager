| **문\*\*\*\*서보안등급 옵션**               | **내용**                                                                                                                                                                                                 | **기본값**                                                                    |
| **문서 보안등급 사용 여부**                   | 문서 보안등급 사용 여부를 설정합니다. **사용함**으로 설정하면 파일별로 보안등급 설정이 가능합니다. 폴더에 보안등급을 지정하면 폴더 내에 저장되어 있는 파일은 자동으로 보안등급이 설정됩니다.                                                                                           | 사용함                                                                        |
| **서비스관리자 기본 취급 권한**                 | 보안등급이 설정된 문서에 대해 서비스관리자에게 부여할 기본 취급 권한을 설정합니다. 다음 12개 권한에 대해 각각 허용 또는 차단 여부를 설정합니다.(1) 목록 보기 (2) 읽기 (3) 저장 (4) 다른 이름 저장 (5) 복사 (6) 이동 (7) 삭제 (8) 클립보드 복사 (9) 화면 캡처 (10) 인쇄 (11) 프린트 워터마크 (12) 스크린 워터마크 | (1)허용 (2)금지 (3)금지 (4)금지 (5)금지 (6)금지 (7)금지 (8)금지 (9)금지 (10)금지 (11)출력 (12)출력 |
| **보안등급이 지정된 폴더로 보안등급문서 이동/복사 시 설정** | 보안등급이 설정되어 있는 문서를 다른 보안등급이 지정되어 있는 폴더로 이동/복사하는 경우에 관한 설정입니다. 다음 3가지 중에서 선택합니다. 1. **기존 문서 보안등급 유지** 2. **폴더에 설정된 보안등급으로 문서 변경** 3. **폴더에 설정된 보안등급과 다르면 이동/복사 차단**                                      | 기존 문서 보안등급 유지                                                              |
| **문서 보안등급 변경 신청 메뉴 일반 사용자 사용 여부**   | 일반 사용자가 문서 보안등급 변경 신청을 할수 있도록 설정할 수 있습니다. **사용함** 설정 시 문서 보안등급 변경 신청 메뉴가 표시되고 사용자는 일반 문서에 보안등급 지정을 신청하거나 기존 보안등급 변경을 신청할 수 있습니다.                                                                       | 사용함                                                                        |
| **문서 보안등급 변경신청 사용자 파일 권한**          | 문서 보안등급을 변경 신청하기 위한 사용자의 파일별 최소 권한을 설정합니다. 사용자가 최소 권한을 충족한 경우에만 문서 보안등급 변경 신청이 가능합니다. 1. **읽기** 2. **삭제** 3. **편집**                                                                                    | 읽기, 삭제, 편집                                                                 |
| **문서 생성 시보안등급 자동 생성 사용 여부**         | 사용자가 문서함에 문서 생성 시 설정한 보안등급으로 자동 생성할지 여부를 설정합니다.                                                                                                                                                        | 사용함                                                                        |
