# 내용 검색 설정

![Info](<../../.gitbook/assets/img_000 (193).png>)메뉴 경로: 시스템 설정 - 설정 홈 연결 - 일반 모듈 설정 - 내용검색

문서중앙화 솔루션의 검색은 2가지로 구분됩니다.

1. 파일명 검색: 검색엔진 사용하지 않고, 서버엔진에서 파일명으로 검색
2. 내용 검색: 별도 검색 엔진 사용

### 내용 검색

|                        |                    |                                                                                            |                                                                                                                                                                                                                           |           |
| ---------------------- | ------------------ | ------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------- |
| **내용 검색 옵션**           |                    |                                                                                            | **내용**                                                                                                                                                                                                                    | **기본값**   |
| **내용 검색 사용 여부**        |                    |                                                                                            | 내용 검색을 위한 별도 검색 엔진 사용여부를 설정합니다. 검색 엔진을 사용하면 문서 내 텍스트를 키워드로 검색할 수 있는 기능을 제공합니다.**사용함**으로 설정하면 검색 엔진으로 내용 검색이 가능하고 **사용 안 함**으로 설정하면 서버 엔진으로 파일명 검색만 가능합니다. 각 검색 화면은 아래와 같습니다. ◾ 검색 엔진을 사용한 내용 검색 화면 ◾ 서버 엔진을 사용한 파일명 검색 화면 | 사용 안 함    |
| **검색 서버**              |                    |                                                                                            | 별도의 검색 서버를 두고 매일 배치 작업을 돌려 이 검색 서버에 검색에 사용될 내용을 업데이트합니다.                                                                                                                                                                  |           |
|                        | **기본 검색**          |                                                                                            | 서버 주소와 포트를 설정합니다.                                                                                                                                                                                                         |           |
| **분산 구성 사용 여부**        |                    | 문서함별로 검색서버를 분산 구성할 수 있습니다. 분산 구성 여부를 설정합니다. 분산 구성을 사용할 경우에는 Manager DB에 개별로 등록하여 사용해야 됩니다. | 사용 안 함                                                                                                                                                                                                                    |           |
| **SSL 사용 여부**          |                    | SSL 사용 여부를 설정합니다. **사용함**으로 설정하면, 검색 서버에 SSL인증서 적용 이후에 검색이 가능합니다.                          | 사용 안 함                                                                                                                                                                                                                    |           |
|                        | **인증서 경로**         | SSL 인증서가 저장되어 있는 경로를 입력합니다.                                                                |                                                                                                                                                                                                                           |           |
| **인증서 비밀번호**           | 인증서 비밀번호를 설정합니다.   |                                                                                            |                                                                                                                                                                                                                           |           |
| **서비스관리자 검색 가능 여부**    |                    |                                                                                            | 서비스관리자의 검색 사용 여부를 설정합니다.**사용함**으로 설정하면 서비스관리자는 모든 문서를 검색할 수 있습니다.                                                                                                                                                         | 사용 안 함    |
| **검색 권한 설정**           |                    |                                                                                            | 사용자의 파일 권한에 따라 검색 결과 목록을 보여줄 범위를 설정합니다. 목록보기 권한만 있을 경우에는 읽기 권한이 없는 파일까지도 검색 결과에 표시됩니다. 1. **목록 보기** 2. **읽기**                                                                                                             | 목록 보기, 읽기 |

|                        | **파일 증분 처리 사용 여부** |                                                                                            | 배치 작업의 결과 파일 업데이트 시 증분 처리 사용 여부를 설정합니다.                                                                                                                                                                                   | 사용함       |

### 키워드 점검

|                  |                                                                                                                                                                                                                    |         |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------- |
| **키워드 점검 옵션**    | **내용**                                                                                                                                                                                                             | **기본값** |
| **키워드 탐지 사용 여부** | 키워드 탐지 주기 사용 여부를 설정합니다.                                                                                                                                                                                            | 사용함     |
| **탐지 주기 설정**     | 아래 주기에서 하나를 선택하고 주기별로 탐지할 상세 일정을 설정합니다. 1. **매일**: 일 간격을 설정합니다.     예) 매 1일마다, 매 2일 마다 1. **매주**: 매주 간격과 요일을 설정합니다.     예) 매1주 마다 화, 수, 목요일에 1. **매월**: 매월 특정 일자를 설정합니다. 말일을 지정할 수 있습니다.             예) 매월 15일, 말일 |         |
| **문서함 선택**       | 키워드 점검을 사용할 문서함을 설정합니다. **+ 폴더선택**을 클릭하여 문서함 및 하위 폴더를 선택할 수 있습니다. 여기서 폴더를 선택하면 관리자 메뉴에 기본 값으로 설정됩니다.                                                                                                               |         |
