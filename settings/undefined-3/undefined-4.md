# 파일 헤더 설정

![Info](<../../.gitbook/assets/img_000 (218).png>)메뉴 경로: 시스템 설정 - 설정 홈 연결 - 고급 설정 - 파일 헤더파일이 헤더를 갖도록 설정하여 추가적인 정보나 기능을 제공할 수 있습니다.**내용검색, 문서태그, 문서보안등급, 문서 만료일** 등의 기능을 사용하기 위해서는 반드시 파일 헤더를 설정해야 합니다.파일 헤더를 사용하는 경우에는 파일명 뒤에 **.cld+** 가 붙어서 저장됩니다.![Notes](<../../.gitbook/assets/img_001 (173).png>)파일 헤더에 관한 자세한 내용은 [**문서함에 저장된 파일의 이름과 구조**](../../system/sw/undefined-4.md)를 참고하세요 .

| ----------------------------------------- | -------------------------------------------------------------------------------------------------------------------- | ------- |
| **파일 헤더 옵션**                              | **내용**                                                                                                               | **기본값** |
| **파일 헤더 사용 여부**                           | 문서중앙화 서버에 저장 시 원본 파일에 헤더 추가 사용 여부를 설정합니다.                                                                            | 사용함     |
| **파일 헤더 정보에 마지막\*\*\*\*Lock 처리 시간 추가 여부** | 파일 헤더 정보에 마지막 Lock 처리시간 추가 여부를 설정합니다. **사용함**으로 설정하면 파일의 Lock이 생성 시 파일 헤더에 **파일의 마지막 접근일** 항목이 Lock 생성 시간으로 업데이트됩니다. | 사용 안 함  |
