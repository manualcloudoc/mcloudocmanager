# 웹페이지 설정

![Info](<../../.gitbook/assets/img_000 (211).png>) 메뉴 경로: 시스템 설정 - 설정 홈 연결 - 고급 설정 - 웹페이지 설정

|                                    |                                                                                                                                                                                                |                                    |
| ---------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------- |
| **웹페이지 설정 옵션**                     | **내용**                                                                                                                                                                                         | **기본값**                            |
| **​** **로그인 전 메인 로고 이미지 등록**       | 로그인 전 메인 화면의 로고 이미지를 설정합니다. 로고 이미지는 메인 화면 좌측 상단에 위치합니다. 권장하는 이미지의 크기는 가로 200px, 세로 80px이며, 이미지 파일 종류는 png 만 가능합니다.**파일 선택** 버튼을 클릭하여 이미지 선택 후 미리 보기가 가능합니다. **적용** 버튼을 클릭해야만 시스템에 실제로 적용됩니다.   |                                    |
| **​** **로그인 후 메인 로고 이미지 등록**       | 로그인 후 메인 로고 이미지를 설정합니다. 로고 이미지는 문서중앙화 웹 페이지 좌측 상단에 위치합니다. 권장하는 이미지의 크기는 가로 200px, 세로 60px이며, 이미지 파일 종류는 png 만 가능합니다.**파일 선택** 버튼을 클릭하여 이미지 선택 후 미리 보기가 가능합니다. **적용** 버튼을 클릭해야만 시스템에 실제로 적용됩니다. |                                    |
| **​** **커스텀 메인 이미지 등록**            | 메인 화면 이미지를 설정할 수 있습니다. 웹페이지 접속 시 보이는 화면입니다. 권장하는 이미지의 크기는 가로 1920px, 세로 1080px이며, 이미지 파일 종류는 jpg 만 가능합니다.**파일 선택** 버튼을 클릭하여 이미지 선택 후 미리 보기가 가능합니다. **적용** 버튼을 클릭해야만 시스템에 실제로 적용됩니다.            |                                    |
| **​** **로고 클릭 시 이동 주소** **​**      | 메인 로고 클릭 시 이동할 메뉴의 주소를 입력합니다. 사용자 웹 페이지에만 적용됩니다.                                                                                                                                               | ​ ‘/’                              |
| **​** **처음 화면 주소**                 | ​ 웹페이지에 처음 접속 시 연결되는 화면 주소를 설정합니다.                                                                                                                                                             | ​ /websvc/index/index\_service.jsp |
| **​** **웹페이지 보안접속 설정**             | ​ 웹페이지 URL을 https로 기본 사용 여부를 설정합니다. **사용함**으로 설정 시 http로 접속하면 https로 URL을 전환합니다.                                                                                                               | ​ 사용 안 함                           |
| **​** **메인 화면 관리자 연락처 표시**         | ​ 접속 웹페이지 메인 화면 중앙 하단에 관리자 연락처 표시 여부를 설정합니다. 1. **표시** 2. **숨김**                                                                                                                               | ​ 표시                               |
| **​** **메인 화면 관리자 이메일 표시**         | ​ 접속 웹페이지 메인 화면 중앙 하단에 관리자 이메일 표시 여부를 설정합니다. 1. **표시** 2. **숨김**                                                                                                                               | ​ 표시                               |
| **​** **웹로그인 비밀번호 저장 사용 여부** **​** | 웹로그인 창에서 비밀번호 저장 기능 사용 여부를 설정합니다. **사용함**으로 설정하면 관리자/사용자 둘 다 적용할 수 있습니다.                                                                                                                       | ​ 사용 안 함                           |
| **​** **사용자 비밀번호 변경 사용 여부**        | 사용자가 웹페이지에서 비밀번호 변경 가능 여부를 설정합니다.**사용함**으로 설정하면 사용자가 웹페이지 **계정관리-비밀번호 변경** 메뉴에서 비밀번호 변경을 할 수 있습니다.**사용 안 함**으로 설정하면 사용자 웹페이지에 비밀번호 변경 메뉴가 표시되지 않아 비밀번호를 변경할 수 없습니다.                            | ​ 사용함                              |
| **​** **사용자 웹페이지 추가 메뉴**           | ​ 사용자 웹 페이지에서 다음 기능을 사용하도록 설정할 수 있습니다. 선택한 메뉴만 사용자 웹 페이지에 표시됩니다. 1. **모바일 앱 설치** 2. **공지사항** 3. **용량 추가 신청**                                                                                   | ​ 모바일 앱 설치, 공지사항, 용량 추가 신청         |
| **​** **관리자 마지막 로그인 정보 알림 사용 여부**  | 관리자 페이지 로그인 시 이전의 마지막 로그인 시간과 로그인 IP정보를 팝업 창에서 보여주는 알림 사용 여부를 설정합니다.                                                                                                                           | ​ 사용함                              |
