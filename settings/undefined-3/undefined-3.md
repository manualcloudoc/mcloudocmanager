## 암호화파일 시스템 설정

문서중앙화 서버에 저장되는 문서는 원본 그대로 또는 암호화해서 저장할 수 있습니다.\
​

|                     |                                                                                                                                 |         |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------- | ------- |
| **암호화파일 시스템 옵션**    | **내용**                                                                                                                          | **기본값** |
| **암호화파일 시스템 사용 여부** | 암호화파일 시스템 사용 여부를 설정합니다. **사용함**으로 설정하면 파일은 암호화되어 저장되고 변환된 파일명 뒤에  **.secure+** 확장자가 붙게 됩니다. **사용 안 함**으로 설정하면 문서는 원본 그대로 저장됩니다. | 사용 안 함  |
| **암호화키 고정값으로 사용**   | 암호화키 고정값 사용 여부를 설정합니다. **사용 안 함**이 기본값이나, 암호화 키파일을 사용하려는 경우에 **사용함**으로 설정합니다. 이 때 암호화 키파일을 서버에 먼저 생성 후 **사용함**으로 설정해야 합니다.      | 사용 안 함  |
