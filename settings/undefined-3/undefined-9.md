# 파일 보관일 설정

![Info](<../../.gitbook/assets/img_000 (220).png>)메뉴 경로: 시스템 설정 - 설정 홈 연결 - 고급 설정 - 파일 보관일 설정\
문서중앙화 서버에 파일 보관일을 설정할 수 있습니다. 옵션별로 **보관 기간**(단위: 일)을 설정하거나 **삭제 안 함** 또는 **보관하지 않음**을 설정할 수 있습니다.

|                          |                                                                     |         |
| ------------------------ | ------------------------------------------------------------------- | ------- |
| **파일 보관일 설정 옵션**         | **내용**                                                              | **기본값** |
| **휴지통 비운 파일 임시 보관기간**    | 휴지통 비운 파일의 임시 보관기간을 설정합니다. 1. **보관기간** (단위: 일) 2. **삭제 안 함**        | 180일    |
| **문서반출 만료파일 임시 보관기간**    | 문서반출 만료 파일의 임시 보관기간을 설정합니다. 1. **보관기간** (단위: 일) 2. **삭제 안 함**       | 삭제 안함   |
| **문서반출 요청파일 임시 보관기간**    | 문서반출 요청 파일의 임시 보관기간을 설정합니다. 1. **보관기간** (단위: 일) 2. **삭제 안 함**       | 삭제 안함   |
| **외부DRM 만료파일 임시 보관기간**   | 외부DRM 만료파일의 임시 보관기간을 설정합니다. 1. **보관기간** (단위: 일) 2. **삭제 안 함**       | 180일    |
| **외부DRM 요청파일 임시 보관기간**   | 외부DRM 요청파일의 임시 보관기간을 설정합니다. 1. **보관기간** (단위: 일) 2. **삭제 안 함**       | 180일    |
| **보안/대외용 웹링크 복사본 보관기간**  | 보안/대외용 웹링크 복사본의 보관기간을 설정합니다. 1. **보관기간** (단위: 일) 2. **보관하지 않음**     | 30일     |
| **문서 보안등급 만료파일 임시 보관기간** | 문서 보안등급 만료파일의 임시 보관기간을 설정합니다. 1. **보관기간** (단위: 일) 2. **삭제 안 함**     | 180일    |
| **프로젝트 버전관리 삭제 작업 보관기간** | 프로젝트 버전관리에서 삭제된 백업의 보관기간을 설정합니다. 1. **보관기간** (단위: 일) 2. **보관하지 않음** | 30일     |

![Warning](<../../.gitbook/assets/img_001 (174).png>)**삭제 안 함**을 사용할 경우 서버 스토리지 공간 부족 문제가 발생할 수 있으니 주의하세요.
