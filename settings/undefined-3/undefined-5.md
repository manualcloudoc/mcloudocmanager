# 휴지통 설정

![Info](<../../.gitbook/assets/img_000 (215).png>)메뉴 경로: 시스템 설정 - 설정 홈 연결 - 고급 설정 - 휴지통

|                             |                                                                                           |         |
| --------------------------- | ----------------------------------------------------------------------------------------- | ------- |
| **휴지통 옵션**                  | **내용**                                                                                    | **기본값** |
| **개인문서함 휴지통 사용 여부**         | 개인문서함 휴지통 사용 여부를 설정합니다. **사용함**으로 설정하면 개인문서함 사용량에 휴지통 용량이 포함됩니다.                          | 사용함     |
| **부서문서함 휴지통 사용 여부**         | 부서문서함 휴지통 사용 여부를 설정합니다. **사용함**으로 설정하면 부서문서함 사용량에 휴지통 용량이 포함됩니다.                          | 사용함     |
| **삭제 시 파일 수정일을 삭제일로 변경 여부** | 삭제 시 휴지통으로 이관되는 파일의 수정일을 삭제일로 변경할 것인지 여부를 설정합니다. 휴지통으로 이동된 삭제 날짜 기준으로 휴지통을 비우기 위해서 변경합니다. | 사용 안 함  |
| **휴지통 비우기 시 파일 임시보관 여부**    | 사용자가 휴지통 비우기를 하더라도 문서중앙화 서버의 임시 보관 영역에 보관할 수 있습니다. 보관 여부를 설정합니다.                          | 사용함     |
