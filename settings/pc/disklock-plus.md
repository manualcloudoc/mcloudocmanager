# DiskLock Plus 설정

메뉴 경로: 시스템 설정 - 설정 홈 연결 - PC 보안 모듈 설정 - DiskLock Plus

### 화면 캡처 금지

|                    |                                      |         |
| ------------------ | ------------------------------------ | ------- |
| **화면 캡처 금지 옵션**    | **내용**                               | **기본값** |
| **화면 캡처 금지 사용 여부** | 화면 캡처를 금지할 애플리케이션을 등록하여 제한하도록 설정합니다. | 사용함     |

|                   |                                     |         |
| ----------------- | ----------------------------------- | ------- |
| **클립보드 복사 제어 옵션** | **내용**                              | **기본값** |
| **클립보드 복사 제어**    | 등록된 애플리케이션 간의 클립보드 복사만 허용하도록 설정합니다. | 사용함     |

### 프린터 워터마크

|                       |                                                                                                                                                                                                                             |          |
| --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- |
| **프린트 워터마크 옵션**       | **내용**                                                                                                                                                                                                                      | **기본값**  |
| **프린트 워터마크 사용 여부**    | 프린트 워터마크에서 이미지 또는 텍스트를 사용하도록 설정합니다.                                                                                                                                                                                         | 사용함      |
| **프린터 워터마크 출력 문서 유형** | 프린트 워터마크를 출력할 문서의 종류를 설정합니다. 1. **보안등급 문서만 출력** 1. **모든 문서 출력** **보안등급 문서만 출력**을 선택한 경우 아래와 같은 추가 작업이 필요합니다. 1) 프린트 워터마크 정책 적용 후2) **일반 모듈 관리-문서보안등급-등급별 사용자 취급 권한 관리-권한 설정** 메뉴에서원하는 보안등급의 프린트 워터마크 항목을 **출력**으로 설정해야 합니다. | 모든 문서 출력 |
