# IP대역 정책 설정

![Info](<../../.gitbook/assets/img_000 (187).png>)메뉴 경로: 시스템 설정 - 설정 홈 연결 - PC 보안 모듈 설정 - IP 대역 정책\
IP대역 정책을 설정하면 지정된 IP대역에서 로그인 시 최우선으로 적용할 정책을 설정할 수 있습니다.

|                           |                                                                                                     |         |
| ------------------------- | --------------------------------------------------------------------------------------------------- | ------- |
| **IP 대역 정책 옵션**           | **내용**                                                                                              | **기본값** |
| **IP 대역\*\*\*\*정책 사용 여부** | **모듈\*\*\*\*통합정책** 또는 **개별정책** 메뉴에서 **IP대역 정책** 메뉴의 사용 여부를 설정합니다. **사용함**으로 설정해야 IP대역 정책 메뉴가 표시됩니다. | 사용함     |
