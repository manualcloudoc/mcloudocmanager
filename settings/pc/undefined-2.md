|                    |                                                                                        |         |
| ------------------ | -------------------------------------------------------------------------------------- | ------- |
| **보안 해제 옵션**       | **내용**                                                                                 | **기본값** |
| **보안 해제 사용 여부**    | 보안 해제 사용 여부를 설정합니다. **사용함**을 선택하면 보안 해제 요청이 승인된 경우 해당 기간동안 프린트 워터마크와 인쇄 제어의 보안이 해제됩니다. | 사용 안 함  |

| **전자결재 연동페이지 너비**  | 전자결재 연동페이지 너비(단위: 픽셀)를 설정합니다.                                                          | 300     |
