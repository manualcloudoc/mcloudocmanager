# 인사 조직 관리 설정

![Info](<../../.gitbook/assets/img_000 (212).png>)메뉴 경로: 시스템 설정 - 설정 홈 연결 - 일반 설정 - 인사 조직 관리

### 조직도

|                            |                                                                                                                                                                             |          |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- |
| **조직도 옵션**                 | **내용**                                                                                                                                                                      | **기본값**  |
| **조직도 연동 사용 여부**           | 조직도 연동 사용 여부를 설정합니다. **사용함**을 선택하면 고객사의 인사관리 시스템에서 조직도 정보를 연동하여 가져와 사용합니다. **사용 안 함**을 선택하면 문서중앙화 시스템에 설정되어 있는 자체 조직도를 사용합니다.                                               | 사용 안 함   |
| **승인권자 연동 사용 여부**          | 승인권자 연동 사용 여부를 설정합니다. **사용함**을 선택하여 연동하면 고객사의 인사관리 시스템에서 승인권자 정보를 가져와 사용합니다. **사용 안 함**을 선택하면 문서중앙화 시스템의 조직도에 설정되어 있는 승인권자를 사용합니다.                                          | 사용 안 함   |
| **부서, 사용자 표시 순서**          | 조직도 편집기에서 한 부서 내 부서와 사용자가 모두 표시되는 경우 부서와 사용자 중 표시 순서를 설정합니다. 1. \*\*부서 상단 표시:\*\*부서를 목록의 상단에 표시하고 사용자는 그 아래에 표시합니다. 2. \*\*사용자 상단 표시:\*\*사용자를 목록의 상단에 표시하고 부서는 그 아래에 표시합니다. | 부서 상단 표시 |
| **직위 선택 항목 표시 여부**         | 권한설정 또는 공유설정 시 권한설정 대상 목록에서 직위 표시 여부를 설정합니다. **사용 안 함**을 선택하면 조직도 편집기에서 직위 선택 항목이 표시되지 않습니다. 따라서 권한설정 시 직위별로 권한설정을 할 수 없습니다.                                                | 사용함      |
| **조직도의 사용자 아이디 마스킹 표시 여부** | 권한설정 또는 공유설정 시 설정 대상 사용자의 아이디를 알 수 없도록 사용자 아이디의 일부를 마스킹 처리(\*로 표시)할 수 있습니다. 마스킹 표시 사용 여부를 설정합니다.                                                                            | 사용 안 함   |

### 사용자

|                           |                                                                                                                                                        |                                  |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ | -------------------------------- |
| **사용자 옵션**                | **내용**                                                                                                                                                 | **기본값**                          |
| **연동 로그인 인증 사용 여부**       | 로그인 시 그룹웨어나 AD등의 ID/패스워드로 연동하여 인증 받을지를 설정합니다.                                                                                                          | 사용 안 함                           |
| **문서중앙화 기본 비밀번호로 로그인 여부** | 연동 로그인 인증을 사용하는 경우에도 문서중앙화에서 설정하는 비밀번호로 로그인 인증 여부를 설정합니다.                                                                                              | 사용함                              |
| **사용자 가입 신청 사용 여부**       | 사용자가 웹에서 직접 가입 신청 사용 여부를 설정합니다. **사용함**으로 설정해야 웹 페이지에 회원가입 버튼이 표시됩니다.                                                                                  | 사용함                              |
| **회원 정보 입력 항목 설정**        | 사용자 가입 신청 사용 시에만 설정합니다. 관리자가 다음 항목 중 웹 페이지에 표시되는 것을 설정하면 사용자는 선택하여 입력(필수 아님) 할 수 있습니다. 1. **휴대전화번호** 2. **사원번호** 3. **생년월일** 4. **성별** 5. **사무실 전화번호** | 휴대전화번호, 사원번호, 생년월일, 성별, 사무실 전화번호 |
| **가입 신청 처리 방식 설정**        | 사용자 가입 신청을 사용할 경우에만 설정합니다. 1. \*\*자동 승인:\*\*자동 가입 처리됩니다. 2. \*\*관리자 승인:\*\*관리자가 승인해야 가입 처리됩니다.                                                         | 관리자 승인                           |

### 승인권자

|                            |                                                                                                                       |                |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------- | -------------- |
| **승인권자 옵션**                | **내용**                                                                                                                | **기본값**        |
| **자가승인 사용 여부**             | 자가 승인 방식은 신청자 스스로 승인을 완료하는 방식으로 자가 승인 기능을 사용하지 않을 경우 자동으로 승인권자 지정 방식이 적용됩니다. 사용 OFF 시 자가 승인 메뉴(하위 메뉴 포함)를 사용할 수 없습니다. | 사용             |
| **사용자에 대한 승인권자 표시 범위**     | 결재 신청 시 표시되는 승인권자 범위를 정할 수 있습니다. 1. **모든 승인권자 표시** 2. **해당 팀 또는 제일 가까운 폴더 승인권자만 표시**                                  | 모든 승인권자 표시     |
| **최상위 승인권\*\*\*\*자 표시 방식** | 승인권자가 본인만 존재할 경우, 본인을 승인권자 목록에 표시할지 여부를 설정합니다. 1. **본인 표시(자가 승인 허용)** 2. **본인 표시 안함(자가 승인 허용 불가)**                    | 본인 표시(자가승인 허용) |
