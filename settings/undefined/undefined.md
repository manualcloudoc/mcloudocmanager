# 윈도우 에이전트 설치 설정

![Info](<../../.gitbook/assets/img_000 (213).png>)메뉴 경로: 시스템 설정 - 설정 홈 연결 - 일반 설정 - 윈도우 에이전트 설치

### 설치 현황

|                   |                                                          |         |
| ----------------- | -------------------------------------------------------- | ------- |
| **설치 현황 옵션**      | **내용**                                                   | **기본값** |
| **설치 현황 사용 여부**   | 윈도우 에이전트의 설치 정보를 수집하고, 사용자별 설치한 디바이스 현황 메뉴 사용 여부를 설정합니다. | 사용함     |
| **설치 상태 확인하는 주기** | 서버가 윈도우 Agent설치 상태를 확인하는 주기(단위: 분)를 설정합니다.               | 180(분)  |

### 오프라인 제품 제거

|                      |                                                                                                                                          |         |
| -------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- | ------- |
| **오프라인 제품 제거 옵션**    | **내용**                                                                                                                                   | **기본값** |
| **오프라인 제품 제거 사용 여부** | 오프라인에서 윈도우 에이전트 제거 가능 여부를 설정합니다. **사용함**을 설정하면 오프라인용 제거 인증코드 발급이 가능합니다. 사용자는 문서 중앙화 서버에 접근이 불가능한 경우 전달받은 인증코드를 이용해 윈도우 에이전트를 제거할 수 있습니다. | 사용함     |
