# 문서 보존 기한 설정 이력 보기

문서에 보존 기한을 신규 설정하거나 기존의 문서 보존 기한을 수정, 또는 해제한 이력을 확인할 수 있습니다.\
1\.   관리자 웹페이지에서 **일반 모듈 관리 - 문서 보존 기한 - 기한 설정 이력**을 선택합니다.\
![](<../../.gitbook/assets/img_000 (141).png>) \
2\.   **기한 설정 이력** 페이지에서 문서 보존 기한 설정 이력 조회를 위한 조건을 선택합니다.

1. **기간 선택**(필수): 문서에 문서 보존 기한을 설정한 날짜입니다.
2. **설정 대상**(필수): 문서 보존 기한 설정 대상을 **파일**과 **폴더** 중 선택합니다.
3. **설정 종류**(필수): 문서 보존 기한을 설정한 작업 종류입니다.
4. **추가**: 문서 보존 기한이 없는 문서에 문서 보존 기한을 설정
5. **수정**: 문서에 설정되어 있는 문서 보존 기한을 변경
6. **해제**: 문서에 설정된 문서 보존 기한을 **기한 없음**으로 변경

3\.   필요 시 검색 조건으로 **파일명, 설정자 이름, 설정자 ID**를 입력합니다. 이때 검색어의 일부만 입력하여 검색할 수 있습니다.

4\.   **검색** 버튼을 클릭하면, 입력한 조건에 해당하는 설정 작업에 대해 **설정일시, 문서함, 설정자 이름(ID), 문서 보존 기한, 파일명, 설정 종류**와 **파일 경로**가 표시됩니다.

1. **설정일시**: 문서 보존 기한이 설정된 일시
2. **문서함**: 문서 보존 기한이 설정된 파일이 위치하는 문서함
3. **설정자 이름(ID**): 문서 보존 기한을 설정한 폴더관리자 또는 서비스관리자의 정보
4. **문서 보존 기한**: 문서 보존 기한의 추가, 수정 시에는 문서에 새로 설정된 문서 보존 기한, 문서 보존 기한을 해제한 경우에는 ‘기한 없음’이 표시됩니다.
5. **파일명**: 문서 보존 기한이 설정된 파일명
6. **설정 종류**: 추가, 수정, 해제 (2번의 설명과 동일)
7. **파일 경로**: 문서 보존 기한이 설정된 파일의 전체 경로
