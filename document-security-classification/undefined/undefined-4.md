# 문서보안등급 설정 이력 보기

문서에 보안등급을 신규 설정하거나 기존의 보안등급을 수정, 또는 해제한 이력을 확인할 수 있습니다.&#x20;

1. 관리자 웹페이지에서 **일반 모듈 관리 - 문서 보안 등급 - 문서 등급 설정 이력**을 선택합니다.&#x20;

<figure><img src="../../.gitbook/assets/img_000 (147).png" alt=""><figcaption></figcaption></figure>

2. **문서 등급 설정 이력** 페이지에서 문서보안등급 설정 이력 조회를 위한 조건을 선택합니다.

* **기간 선택**(필수): 문서에 보안등급을 설정한 날짜입니다.
* **설정 대상**(필수): 보안등급 설정 대상을 **파일**과 **폴더** 중 선택합니다.
* **보안 등급**(필수): 보안등급의 추가, 수정 시에는 문서에 새로 설정된 보안등급, 보안등급을 해제한 경우에는 해제 전의 보안등급에 해당합니다. 완전히 삭제된 등급은 표시되지 않으며, 등급 삭제 시 편입된 경우에는 삭제된 등급명이 ‘편입됨’과 함께 표시됩니다. (예: ‘5등급 편입됨’)
* **설정 종류(필수)**: 보안등급을 설정한 작업 종류입니다.

> - **추가**: 문서에 보안등급을 신규 설정
> - **수정**: 문서에 설정되어 있는 보안등급을 변경
> - **해제**: 문서에 설정된 보안등급을 해제

3. 필요 시 검색 조건으로 **파일명**, **설정자 이름**, **설정자 ID**를 입력합니다. 이때 검색어의 일부만 입력하여 검색할 수 있습니다.
4. **검색** 버튼을 클릭하면, 입력한 조건에 해당하는 설정 작업에 대해 \*\*설정일시, 보안 등급, 파일명, 문서함, 설정 종류, 만료일, 설정자 이름(ID)\*\*와 **파일 경로**가 표시됩니다.

* **설정일시**: 보안등급이 설정된 일시
* **보안 등급**: 보안등급의 추가, 수정 시에는 문서에 새로 설정된 보안등급명, 보안등급을 해제한 경우에는 해제 전의 보안등급명이 표시됩니다. 보안등급이 삭제된 경우에는 삭제된 등급 코드가 ‘(없음)’과 함께 표시됩니다. (예: ‘05(없음)’)
* **파일명**: 보안등급이 설정된 파일명
* **문서함**: 보안등급이 설정된 파일이 위치하는 문서함
* **설정 종류**: 추가, 수정, 해제 (2번의 설명과 동일)
* **만료일**: 보안등급 설정이 만료되는 일자
* **설정자 이름(ID)**: 보안등급을 설정한 폴더관리자 또는 서비스관리자의 정보
* **파일 경로**: 보안등급이 설정된 파일의 전체 경로
