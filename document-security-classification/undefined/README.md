# 문서보안등급 및 보안등급문서 관리하기

문서보안등급 기능은 중앙문서함에 저장된 문서에 보안등급을 부여하여 보안상 특별한 취급이 필요한 중요 문서를 체계적으로 관리하고 보호하는 기능입니다.&#x20;

{% hint style="warning" icon="square-poll-horizontal" %}
문서보안등급에 대한 기본적인 설명은 사용자 매뉴얼 – [**문서보안등급 소개**](https://app.gitbook.com/s/vQ0BiQsGY4PT08D7Nfay/undefined-4)를 참고합니다.
{% endhint %}

문서보안등급 기능을 활용하기 위해서는 관리자(정보보호관리자 또는 서비스관리자)가 보안등급 구조를 설정하고 보안등급별 취급 권한을 부서 및 사용자에 따라 차별화하여 부여하는 등, 다음과 같은 사전 설정 작업을 수행해야 합니다.
