# 문서보안등급 구조 및 취급 권한을 최신 정보로 즉시 업데이트하기

관리자는 필요 시 최신 문서보안등급 정보와 사용자 취급 권한 정보를 모든 파일 서버 캐시에 반영할 수 있습니다.

1\.   관리자 웹페이지에서 **일반 모듈 관리 - 문서 보안 등급 - 캐시 업데이트**를 선택합니다.\
![](<../../.gitbook/assets/img_000 (144).png>)

2\.   필요에 따라 **등급 구조 정보 캐시 업데이트** 또는 **등급별 권한 정보 캐시 업데이트**의 **설정** 버튼을 클릭하여 캐시 업데이트를 실행합니다. 업데이트가 완료되면 “갱신되었습니다”라는 알림 창이 표시됩니다.

1. **등급 구조 정보 캐시 업데이트**

문서보안등급 구조를 설정한 후, 정의된 보안등급 적용이 정확하지 않다고 의심되는 경우에 사용합니다. 실행 시 데이터베이스의 최신 문서보안등급 구조 정보가 모든 파일 서버의 **문서보안등급 등급정의 캐시**에 즉시 반영됩니다.

1. **등급별 권한 정보 캐시 업데이트**

사용자의 문서 취급 권한을 설정한 후, 설정된 등급별 권한 적용이 정확하지 않다고 의심되는 경우에 사용합니다. 실행 시 데이터베이스의 최신 문서보안등급별 취급 권한 정보가 모든 파일 서버의 **문서보안등급 취급권한 캐시**에 즉시 반영됩니다.&#x20;

![Notes](<../../.gitbook/assets/img_001 (135).png>)캐시 파일에 대한 자세한 내용 및 업데이트 절차는 \*\*[파일서버 캐시 종류 및 갱신 방법](../../system/undefined-1/undefined-1.md)\*\*을 참고합니다.
