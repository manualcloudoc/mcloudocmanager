# 문서보안등급 변경 신청 처리 이력 보기

관리자는 사용자가 문서의 보안등급 변경을 신청한 건에 대해 폴더관리자가 승인 또는 반려 처리한 이력을 확인할 수 있습니다.

1. 관리자 웹페이지에서 **일반 모듈 관리 - 문서 보안 등급 - 문서 등급 변경 이력**을 선택합니다.

<figure><img src="../../.gitbook/assets/img_000 (146).png" alt=""><figcaption></figcaption></figure>

2. **문서 등급 변경 이력** 페이지에서 변경 신청 이력 조회를 위한 조건을 설정합니다.

* 기간 선택(필수): 사용자가 문서의 보안등급 변경을 신청한 날짜입니다.
* 결재 방식(필수): 변경 신청 시 적용된 결재 방식을 선결제, 후결제 중에서 선택합니다.
* 결재 상태(필수): 변경 신청 건의 처리 상태입니다.

> - **대기**: 폴더관리자의 결재를 대기 중인 상태
> - **승인**: 사용자의 신청을 폴더관리자가 승인함
> - **반려**: 사용자의 신청을 폴더관리자가 반려함
> - **취소**: 폴더관리자가 결재하기 전에 사용자가 신청했던 건을 취소함

3. 필요 시 **검색 조건**으로 **신청자 ID/이름**, **승인권자 ID/이름**을 입력합니다. 이때 검색어의 일부만 입력하여 검색할 수 있습니다.
4. **검색** 버튼을 클릭하면, 입력한 조건에 해당하는 변경 신청 건의 **신청일시, 제목, 결재 방식, 결재 상태, 결재 승인/반려 일시, 신청자 이름/ID, 승인권자 이름/ID**가 표시됩니다.
5. 상세한 내역을 확인하기 위해서 **제목** 항목을 클릭하면 **상세보기** 페이지에서 신청자가 작성한 신청 **내용**, 승인권자가 작성한 **승인/반려 사유**, 파일이 위치한 **문서함**을 추가로 확인할 수 있습니다. 또한 페이지 하단의 **등급 변경 신청 목록**에서 변경 신청한 파일의 경로, 변경 전/후의 **등급(등급명)**, **만료일**을 확인할 수 있습니다.
