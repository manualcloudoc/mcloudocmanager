# 애플리케이션 카테고리 자동등록 설정하기

애플리케이션 등록 시 조건에 따라 특정 카테고리로 자동 분류되도록 설정할 수 있습니다.

**PC 보안 모듈 관리 - DiskLock - 로컬저장금지 - 애플리케이션 카테고리 자동등록** 메뉴에서 자동분류 설정을 추가, 조회, 변경, 삭제하고, 설정 내용을 즉시 실행할 수 있습니다.

![](<../../.gitbook/assets/img_000 (135).png>)

#### <mark style="color:$primary;">자동분류 추가하기</mark>

새로운 자동분류 설정을 등록하는 방법은 다음과 같습니다.

1. 애플리케이션 카테고리 자동등록 페이지 우측의 **추가** 버튼을 클릭하면, 다음의 페이지가 표시됩니다.

![](<../../.gitbook/assets/img_001 (127).png>)

2. 다음의 자동분류 설정 항목을 필수적으로 입력합니다.

* **등록명**: 자동분류의 이름을 지정합니다.
* **카테고리**: **선택** 버튼을 클릭하여 애플리케이션 카테고리 트리에서 분류될 카테고리를 선택하고, **확인** 버튼을 클릭합니다. 이때 애플리케이션 파일명으로 검색하여, 해당 애플리케이션에 대해 기존에 설정된 카테고리를 선택할 수 있습니다.

3. 다음 항목 중 자동분류 시 사용할 옵션을 선택하여 좌측 버튼을 활성화한 후, 필요한 항목 값을 입력합니다. 이때 적어도 한 가지 항목은 반드시 선택해야 합니다.

* **애플리케이션 파일명**: 자동분류를 진행할 애플리케이션 파일명을 지정합니다. 이때 와일드카드 문자(\*,?)를 사용하여 설정할 수 있습니다.
* **제품명**: 애플리케이션의 제품 이름을 사용하여 자동분류 대상을 설정할 수 있습니다.
* **회사명**: 애플리케이션의 제작사 이름을 사용하여 자동분류 대상을 제한할 수 있습니다.

{% hint style="danger" %}
**제품명, 회사명**을 사용할 경우, 반드시 DiskLock 콘솔에 표시된 이름을 입력해야 합니다.
{% endhint %}

* **애플리케이션 경로**: 특정 경로에 위치하는 애플리케이션으로 자동분류 대상을 제한할 수 있습니다. 이때 해당 경로를 직접 입력하거나, 예약 폴더를 선택한 후 하위 경로를 추가할 수 있습니다.  예약 폴더의 예는 다음과 같습니다.

> - Windows 폴더: C:\Windows, C:\WinNT
> - 시스템 폴더: C:\Windows\System32, C:\WinNT\System32

{% hint style="success" icon="lightbulb-exclamation-on" %}
예약 폴더를 사용한 **애플리케이션 경로** 입력 방법은 다음과 같습니다.\
“C:\Windows\System32\Microsoft”로 경로를 지정할 경우 드롭다운 박스에서 **시스템 폴더**를 선택 후, 텍스트 박스에 “\Microsoft” 와 같이 표시되도록 “\Microsoft”를 덧붙입니다.
{% endhint %}

* **버전**: 애플리케이션의 특정 버전을 자동분류 대상을 제한할 수 있습니다. 이때 버전이 입력한 문자열과 정확히 일치하는 경우로 한정하거나, 입력한 문자열로 시작하는 버전을 사용하도록 선택할 수 있습니다.
* **즉시 분류**: 애플리케이션이 등록되자마자 즉시 설정된 카테고리로 분류됩니다. 즉시 분류를 사용하지 않을 경우에는 정해진 주기(예: 10분)마다 자동 분류 작업이 수행됩니다.

{% hint style="danger" %}
즉시 분류되는 애플리케이션이 많을 경우, 서버에 부하가 발생할 수 있으므로 주의가 필요합니다.
{% endhint %}

#### <mark style="color:$primary;">자동분류 검색하기</mark>

이미 등록되어 있는 자동분류 설정을 검색 조건에 따라 검색하여 조회할 수 있습니다.

1. 페이지 상단에서 검색 조건을 지정합니다. **카테고리 이름**, **애플리케이션 파일명**, **회사명**, **제품명**, **버전** 중 검색할 항목을 선택하고, 우측에 검색어를 입력합니다. 이때 검색어의 일부만 입력하여 검색할 수 있습니다.
2. **검색** 버튼을 클릭하면, 검색 조건에 해당하는 **자동분류의 등록명**, **파일 이름**, **회사명**, **제품명**, **버전**, **즉시 분류 여부**, **애플리케이션 파일 경로**, **카테고리 경로**가 하단에 표시됩니다.

#### <mark style="color:$primary;">자동분류 변경하기</mark>

검색 결과 표시된 자동분류 목록 중, 변경을 원하는 설정의 **등록명**을 클릭하여 등록 내용을 변경할 수 있습니다. 자동분류 설정 항목을 변경하는 방법은 **자동분류 추가하기**에서와 같습니다.

#### <mark style="color:$primary;">자동분류 삭제하기</mark>

자동분류 목록에서 삭제할 설정의 우측 휴지통 아이콘을 클릭하여 해당 등록 내용을 삭제할 수 있습니다.

#### <mark style="color:$primary;">자동분류 사용 여부 변경하기</mark>

등록되어 있는 자동분류 설정을 건별로 사용하거나, 사용하지 않도록 설정할 수 있습니다. 현재 서버에서 사용하도록 설정된 자동분류의 경우 자동분류 목록에서 좌측의 체크박스가 체크되어 표시됩니다. 자동분류 사용 여부를 변경하기 위해서는, 변경하려는 설정의 좌측 체크박스를 **체크**(사용) 또는 **체크 해제**(미사용)한 후 **사용 여부 적용** 버튼을 클릭합니다.

#### <mark style="color:$primary;">자동분류 즉시 실행하기</mark>

자동분류 목록 우측 상단의 **자동등록 실행**버튼을 클릭하면, 사용 설정된 자동분류 기준에 따라 **미분류** 애플리케이션에 대한 카테고리 분류를 즉시 실행할 수 있습니다.
