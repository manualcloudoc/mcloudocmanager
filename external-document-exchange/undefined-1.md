# 대외용 웹링크 복사 및 대외용 링크메일 이력 보기

대외용 링크메일, 대외용 웹링크의 생성 및 결재 이력을 기간, 수신자 유형, 승인 처리 상태, 사용자, 승인권자 및 파일 종류별로 검색하여 확인할 수 있습니다.&#x20;

1. **일반 모듈 관리 - 대외문서교환 - 대외용 - 링크메일/웹링크 - 이력 보기**를 클릭합니다.
2. 조회를 원하는 **기간**, **링크 종류**, **수신자 유형**, **작업 상태**를 선택합니다.

* **링크 종류**: 이력 조회를 원하는 링크 종류를 **전체, 대외용 링크메일, 대외용 웹링크** 중 선택합니다.
* **수신자 유형**

> - 사용자가 대외용 링크메일, 웹링크 생성 요청 시 선택한 \*\*수신자 유형(내부/계열사/협력사 직원)\*\*별로 조회가 가능합니다.
> - **유형 없음**:  구 버전 대외용 링크메일, 웹링크 생성 이력의 경우, 수신자 유형이 **유형 없음**으로 분류됩니다.

* **작업상태**

> - **대기**: 승인이 필요한 대외용 링크메일, 웹링크 생성 요청 후, 승인권자의 승인을 기다리는 상태입니다.
> - **승인**: 승인이 필요한 대외용 링크메일, 웹링크 생성 요청에 대해 승인권자가 승인을 완료한 상태입니다.
> - **반려**: 승인이 필요한 대외용 링크메일, 웹링크 생성 요청을 승인권자가 반려한 상태입니다.
> - **취소**: 선결재 승인이 필요한 대외용 링크메일, 웹링크 생성 요청을 결재 전에 사용자가 취소한 상태입니다.
> - **완료**: 승인이 필요 없는 대외용 링크메일, 웹링크 생성 요청에 대해 링크 생성이 완료된 상태입니다.

3. 필요시 **검색 조건**으로 **생성자 ID/이름, 승인권자 ID/이름, 제목, 부서명, 파일 이름, 파일 확장자명**을 입력합니다. 이때 검색어의 일부만 입력하여 검색할 수 있습니다. 단, **파일 확장자명**으로 검색 시에는 입력한 문자열이 확장자와 일치해야 검색이 가능합니다.

* **생성자 ID/이름**: 대외용 링크메일, 웹링크 생성을 요청한 사용자별로 검색할 수 있습니다.
* 검색 조건을 입력하지 않을 경우 전체 대외용 링크메일, 웹링크 생성 이력이 표시됩니다.

4. 검색 버튼을 클릭하면, 입력한 조건에 해당하는 사용 이력의 생성일시, 링크 종류, 수신자 유형, 제목, 생성자 이름/ID, 부서명, 승인권자 이름/ID, 작업 상태가 표시됩니다.

* **생성일시**: 대외용 링크메일, 웹링크가 생성된 일시가 표시됩니다.
* **제목**: 대상 파일의 이름이 표시됩니다. 복수 개의 파일에 대해 대외용 링크메일, 웹링크 생성이 요청된 경우, “대표 파일명(외 n개)”와 같이 표시됩니다.
* 조회한 이력을 엑셀 파일로 저장하려면, 화면 우측 상단에 있는 <img src="../.gitbook/assets/img_000 (154).png" alt="" data-size="original"> \[엑셀 다운로드] 버튼을 클릭합니다.

![](<../.gitbook/assets/img_001 (143).png>)

5. 표시된 이력 목록에서 상세보기를 원하는 작업 이력의 **제목**을 클릭하면 해당 작업의 상세 정보를 확인할 수 있습니다.

* **상세보기** 페이지에서 추가적으로 수신자 **인증 방식**, **OTP** **인증 이메일** 주소(이메일 OTP 인증 사용 시), **승인 방식** 및 **승인신청 사유**, **다운로드 유효기한**, **다운로드 가능 횟수**, **승인/반려 사유**(승인/반려 시)를 확인할 수 있습니다.
* 하단의 **파일 목록(다운로드 정보)**&#xC5D0; 대상 파일 각각의 **파일명**, **파일 경로**, **크기** 및 **다운로드 횟수**(사용자가 설정한 총 다운로드 가능 횟수 대비 수신자가 다운로드한 횟수)가 표시됩니다.
* 페이지 우측 하단의 **목록** 버튼을 클릭하면 이력 보기 페이지로 돌아갑니다.
