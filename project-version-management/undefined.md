# 프로젝트 버전 관리 폴더 목록 보기

관리자는 버전 관리가 진행되고 있는 프로젝트 폴더 목록을 문서함별로 확인할 수 있습니다. 버전 관리 등록이 삭제된 폴더는 목록에서 제외됩니다.

1. **백업 모듈 관리 - 프로젝트 버전 관리 - 백업 폴더 현황**을 클릭합니다.

![](<../.gitbook/assets/img_000 (180).png>)

2. 검색 조건을 입력하고 **검색**을 클릭합니다.

* **문서함(필수)**: 전체 문서함을 선택하거나 일부 문서함을 한 개 이상 선택할 수 있습니다. 기본적으로 선택 가능한 모든 문서함이 표시되며, 검색에서 제외할 문서함을 문서함 이름 우측의 X표시를 클릭하여 삭제합니다. **문서함 선택** 버튼을 클릭하면 아래와 같은 ‘**문서함 선택**’ 창에서 문서함을 선택할 수 있습니다.

<div align="left"><figure><img src="../.gitbook/assets/img_001 (165).png" alt=""><figcaption></figcaption></figure></div>

* 기타 검색 조건(선택):

> - **프로젝트명**: 프로젝트 버전 관리 신청 시 작성한 프로젝트 이름
> - **프로젝트 폴더 경로**: 백업 폴더가 생성된 프로젝트 폴더 경로
> - **신청자 ID/이름**: 프로젝트 버전 관리를 신청한 사람의 ID/이름

3. 목록에서 문서함, 프로젝트 폴더 경로, 프로젝트명, 신청자 ID 및 이름, 백업 폴더가 생성된 생성일시를 확인합니다.
