# 빅데이터 통계

​\
빅데이터 통계 기능은 중앙문서함의 문서 현황, 반출 현황, 파일 작업 현황 등 고객사의 문서 생산 및 사용과 관련된 데이터를 제공합니다. 빅데이터 통계 기능을 사용하면 표, 그래프, 차트 등의 시각 자료를 통해 회사의 문서 현황과 파일의 생산/소비 패턴을 파악하고, 문서 반출을 모니터하여 이상 징후 발견 시 보안 문제에 대비할 수 있습니다. 또한 수집된 데이터의 다운로드가 가능하므로, 고객사의 필요에 따라 가공하여 회사의 조직 운영과 보안 관리를 위한 인사이트를 도출하는 데 활용할 수 있습니다.

![Notes](<../.gitbook/assets/img_000 (102).png>)빅데이터 통계는 빅데이터 서비스 구매 시 사용 가능한 기능입니다.

## 빅데이터 통계 제공 기능

### 문서 현황 통계 보기

중앙문서함의 문서 현황을 날짜별로 수집하여 제공하는 기능입니다. 지정된 날짜를 기준으로 특정 문서함 또는 문서함 내 특정 폴더에 저장된 문서의 수량과 전체 용량을 확인할 수 있습니다. 또한 월별/연도별로 문서함에 대한 전체 문서수량과 문서 용량 확인 기능을 제공하여 문서 생산 흐름을 파악할 수 있습니다.​\
​자세한 내용은 [**빅데이터 – 문서 현황 통계 보기**](undefined-2.md)​​를 참고하세요.\
​\
![](<../.gitbook/assets/img_001 (95).png>)  &#x20;

### 문서 반출 통계 보기

중앙문서함의 문서 반출 현황을 날짜별로 수집하여 제공하는 기능입니다. 선택한 기간 동안에 특정 문서함 또는 문서함 내 특정 폴더에서 발생한 문서 반출 현황을 확인할 수 있습니다. 특히, 문서 반출 신청 건수와 파일 수, 파일 용량의 경우 입력한 기간 내의 일자별 변동 추이를 그래프로 확인할 수 있어, 대량 문서 반출과 같은 이상 징후를 발견하고 보안 위협을 탐지하는데 활용할 수 있습니다.\
​\
자세한 내용은 [**빅데이터 – 문서 반출 통계 보기**](undefined-1.md)를 참고하세요.\
​\
![](<../.gitbook/assets/img_002 (80).png>)

### 파일 작업 통계 보기

중앙문서함에서 발생한 파일 작업에 대한 통계를 날짜별로 수집하여 제공하는 기능입니다. 선택한 기간 동안에 특정 문서함에서 발생한 업로드, 다운로드, 삭제 등의 파일 작업 통계(항목별 합계)를 확인할 수 있습니다. 파일 작업에 대해 부서 내 사용자별 통계와 사용자 순위를 차트로도 확인할 수 있어 회사 내 파일의 생산(업로드)과 소비(다운로드) 패턴을 분석하는데 유용하게 활용할 수 있습니다.\
​\
