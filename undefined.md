# 보안뷰어 사용을 위한 문서변환 시스템 소개

**보안뷰어**는 중앙문서함의 문서를 열람하거나 수신한 링크를 통해 문서 열람 시, 문서를 PDF 형식으로 변환한 후에 다운로드 없이 스트리밍으로 열람하는 기능으로, 문서 유출의 가능성을 원천적으로 차단하는 가장 안전한 문서 열람 방법입니다.

{% hint style="warning" icon="square-poll-horizontal" %}
보안뷰어에 대한 자세한 사항은 **사용자 매뉴얼**의 [**보안뷰어 소개**](https://app.gitbook.com/s/vQ0BiQsGY4PT08D7Nfay/secureviewer)를 참고하세요.
{% endhint %}

보안뷰어를 사용하기 위해서는 문서변환 서버가 구축되어야 하며, 전제 조건은 다음과 같습니다.

* 솔루션이 제공하는 **문서변환 서버** 라이선스 보유
* 고객사가 MS Office 와 아래한글에 대한 전직원 라이선스 보유
* 문서 변환 서버를 구축할 시스템에 MS Office, 아래한글 프로그램 설치

아래 그림은 문서변환 시스템의 동작 과정을 나타낸 것입니다. 여기서 문서변환 서버는 문서중앙화 서버와 별도로 Windows 환경에 구축됩니다.

보안뷰어가 적용되는 환경에서 사용자가 문서 열람 시, 문서중앙화 서버는 해당 파일을 문서변환 서버에 전송합니다.  문서변환 서버는 이를 PDF 형식으로 변환하여 문서중앙화 서버에 다시 전달하고, 문서중앙화 서버가 변환된 PDF 파일을 사용자의 브라우저에 스트리밍 방식으로 출력하게 됩니다.
