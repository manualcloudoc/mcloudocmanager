# 개인정보 검출 이력 보기

관리자는 개인정보가 검출된 파일들의 검출 이력을 검색 및 조회할 수 있습니다.&#x20;

1. **일반 모듈 관리 – 개인정보보호 – 검출 이력**을 클릭합니다.

<figure><img src="../.gitbook/assets/img_000 (175).png" alt=""><figcaption></figcaption></figure>

2. 다음과 같이 개인정보 **검출 파일**을 검색할 수 있습니다.

* 조회를 원하는 **기간**, **문서함**종류, **검출 항목, 검색 조건**을 선택/입력한 후 **검색** 버튼을 클릭합니다.
* **문서함 선택** 버튼을 클릭하면 체크 박스 형태로 표시되는 전체 문서함 중 개인정보 검출 파일을 검색할 문서함을 선택하거나 문서함 이름으로 키워드 검색하여 추가할 수 있습니다.  기본값은 전체 문서함 선택입니다.

<figure><img src="../.gitbook/assets/img_001 (161).png" alt=""><figcaption></figcaption></figure>

* **검색 조건**:  필요시 검색 조건으로 **파일명, 파일 경로, 파일 생성자 이름, 파일 생성자 ID**를 입력합니다. 이때 검색어의 일부만 입력하여 검색할 수 있습니다. **검색 조건**을 입력하지 않을 경우 전체 검출 파일 목록이 표시됩니다.

3. 지정된 조건에 해당하는 검출 이력 목록이 표시됩니다. 개인정보가 검출된 파일의 **검출일시, 문서함, 파일 생성자 이름(ID), 파일명, 검출 항목(검출 개수), 파일 생성일, 파일 경로**를 확인할 수 있습니다.

* 목록에 표시된 검출 항목(검출 개수) 내용을 클릭하면 ‘**상세보기**’ 창에서 검출된 내용을 확인할 수 있습니다.
