# 검출할 개인정보 패턴의 사용 여부 설정하기

관리자는 개인정보보호 기능 사용 시 개인정보 검출에 사용될 개인정보 유형(검출 패턴)을 다음과 같이 선택할 수 있습니다.\
​\
1\.   **일반 모듈 관리 – 개인정보보호 – 검출 패턴 관리**를 클릭합니다.\
​\
&#x20;![](<../.gitbook/assets/img_000 (179).png>)\
2\.   다음의 개인정보 검출 패턴별로 사용 여부(사용/사용 안 함)를 설정합니다.

> •  **주민번호, 휴대폰번호, 외국인주민번호, 법인등록번호, 사업자등록번호, 카드번호 16자리, 여권번호, 구 여권번호, 이메일, 전화번호**
