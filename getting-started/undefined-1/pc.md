# 관리자 빠른 시작 가이드 - PC 보안을 위한 필수 설정하기

DiskLock, DiskLock Plus 모듈이 제공하는 PC 보안 기능을 사용하기 위해 관리자는 다음과 같은 작업을 필수적으로 수행해야 합니다.&#x20;

1. **전사에 기본 통합 정책 적용**
2. **프린트 워터마크로 사용할 이미지와 텍스트 등록**
3. **인쇄를 허용할 프린터 등록**

다음 가이드를 따라 위의 3가지 작업을 수행하면 기본 설정이 적용된 상태로 PC 보안 기능을 사용할 수 있게 됩니다.&#x20;

### <mark style="color:$primary;">기본 통합 정책 적용하기</mark>

기본 통합 정책을 적용하는 방법은 다음과 같습니다.

1. **PC 보안 모듈 관리 – 모듈 통합정책 적용 – 정책 적용** 메뉴를 선택합니다.  &#x20;

<figure><img src="../../.gitbook/assets/img_000 (158).png" alt=""><figcaption></figcaption></figure>

2. 좌측의 조직도에서 정책을 적용할 부서 또는 사용자를 선택합니다. 여기서는 전사에 정책을 적용하기 위해 사이트명(위 그림 예시에서는 **MYCOMPANY**)을 클릭합니다.
3. 우측의 정책 목록에서 적용할 정책을 선택합니다. 여기서는 기본 정책인 **00001.기본정책**을 선택합니다.&#x20;
4. 우측 상단의 **적용** 버튼을 클릭하여 시스템에 변경 사항을 반영합니다.

기본 통합 정책은 로컬 저장 금지, 화면 캡처 금지 등 DiskLock과 DiskLock Plus 모듈의 주요 기능별 기본 정책으로 구성됩니다. 해당 정책이 적용되면, 문서중앙화 에이전트가 실행된 윈도우 PC에서는 각 기능별 정책에 따라 사용이 제한됩니다.

각 기능별 기본 정책을 이해하고, 필요에 따라 다른 정책을 설정하려면 [**관리자 빠른 시작 가이드 – PC 보안 정책 이해 및 응용하기**](pc-1.md)를 참고합니다.

### <mark style="color:$primary;">워터마크 이미지/텍스트 등록하기</mark>

사용자에게 적용된 통합 정책 설정에 따라, 사용자 PC에서 인쇄 시 이미지 또는 텍스트 형태로 워터마크가 함께 출력될 수 있습니다. 기본 통합 정책을 적용하면 텍스트 워터마크가 출력됩니다.따라서, 관리자는 고객사에서 사용할 워터마크의 이미지와 텍스트를 다음과 같이 사전에 등록해야 합니다.

1. **PC 보안 모듈 관리 – DiskLock Plus – 프린트 워터마크 – 정책 관리** 메뉴를 선택합니다.

<figure><img src="../../.gitbook/assets/img_002 (126).png" alt=""><figcaption></figcaption></figure>

2. **이미지 등록** 항목의 **파일 선택** 버튼을 클릭하여 워터마크로 사용할 이미지 파일을 선택한 후, **열기**를 클릭합니다. 이미지 파일은 해상도가 300dpi 이상인 png 파일이어야 합니다.
3. **이미지 등록** 항목의 **텍스트 내용** 입력란에 워터마크로 사용할 텍스트를 입력합니다.
4. **백그라운드 설정** 하단의 적용 버튼을 클릭합니다.

### <mark style="color:$primary;">프린터 등록하기</mark>

기본 통합 정책이 적용되면 등록된 프린터에서만 인쇄가 가능해지므로, 관리자가 사전에 인쇄를 허용할 프린터를 다음과 같이 등록해야 합니다.

1. **PC 보안 모듈 관리 – DiskLock Plus – 인쇄 제어 – 프린터 등록/제거** 메뉴를 선택합니다.&#x20;

<figure><img src="../../.gitbook/assets/img_003 (109).png" alt=""><figcaption></figcaption></figure>

2. **프린터 등록/제거** 페이지 우측 상단의 **등록** 버튼을 클릭하면 다음과 같이 ‘프린터 등록’ 창이 팝업됩니다.    &#x20;

<div align="left"><figure><img src="../../.gitbook/assets/img_004 (85).png" alt="" width="395"><figcaption></figcaption></figure></div>

* **프린터명**(필수), **프린터 드라이버**(필수), **프린터 IP**를 입력한 후 **확인**을 클릭하면 입력한 정보가 등록됩니다.
* **프린터명**: 프린터를 식별할 이름을 지정합니다.
* **프린터 드라이버**: 윈도우 PC의 프린터 및 스캐너 설정에서 해당 프린터의 **프린터 속성**에 표시되는 **모델명**을 입력합니다.
